Lines and Planes

Chapter 1: Vectors and the Geometry of Space Lesson 5 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to write equations for lines and planes in three-dimensional space.

In one sentence: Lines and planes in 3D can be described using vectors, points, and equations.

💡 The Idea

Equations of Lines

A line can be described in several ways:

Vector Equation

r = r₀ + tv

Where r₀ is a point on the line and v is the direction vector.

Parametric Equations

x = x₀ + at

y = y₀ + bt

z = z₀ + ct

Symmetric Equations

(x - x₀)/a = (y - y₀)/b = (z - z₀)/c

Equations of Planes

A plane can be described in several ways:

Point-Normal Form

n · (r - r₀) = 0

Where n is the normal vector and r₀ is a point on the plane.

General Form

ax + by + cz = d

Where ⟨a, b, c⟩ is the normal vector.

Finding the Normal Vector

If you have two vectors in the plane, the normal vector is:

n = u × v

Distance from a Point to a Plane

The distance from point P to the plane ax + by + cz = d is:

D = |ax₀ + by₀ + cz₀ - d| / √(a² + b² + c²)

Why This Matters

Understanding lines and planes is essential for 3D geometry, computer graphics, physics, and engineering. These concepts form the foundation for more advanced topics.

📝 Example

Example 1: Finding the Equation of a Line

Problem: Find the equation of the line through (1, 2, 3) with direction vector ⟨2, -1, 4⟩

Step 1: Write the vector equation

r = ⟨1, 2, 3⟩ + t⟨2, -1, 4⟩

Step 2: Write the parametric equations

x = 1 + 2t
y = 2 - t
z = 3 + 4t

Step 3: Write the symmetric equations

(x - 1)/2 = (y - 2)/(-1) = (z - 3)/4

Example 2: Finding the Equation of a Plane

Problem: Find the equation of the plane through (1, 2, 3) with normal vector ⟨4, 5, 6⟩

Step 1: Use the point-normal form

n · (r - r₀) = 0
⟨4, 5, 6⟩ · ⟨x - 1, y - 2, z - 3⟩ = 0

Step 2: Expand

4(x - 1) + 5(y - 2) + 6(z - 3) = 0
4x - 4 + 5y - 10 + 6z - 18 = 0
4x + 5y + 6z = 32

Example 3: Finding the Distance from a Point to a Plane

Problem: Find the distance from (1, 2, 3) to the plane 2x + 3y + 4z = 10

Step 1: Use the distance formula

D = |2(1) + 3(2) + 4(3) - 10| / √(2² + 3² + 4²)
= |2 + 6 + 12 - 10| / √(4 + 9 + 16)
= |10| / √29
= 10/√29 ≈ 1.86

📋 How to Solve Most Questions

Step-by-Step: Finding the Equation of a Line

  1. Identify a point — Find a point on the line
  2. Find the direction vector — Determine the direction
  3. Write the vector equation — r = r₀ + tv
  4. Write parametric equations — x = x₀ + at, etc.
  5. Write symmetric equations — (x - x₀)/a = (y - y₀)/b = (z - z₀)/c

Step-by-Step: Finding the Equation of a Plane

  1. Identify a point — Find a point on the plane
  2. Find the normal vector — Use given vector or cross product
  3. Write point-normal form — n · (r - r₀) = 0
  4. Expand to general form — ax + by + cz = d

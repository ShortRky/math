Equations of Lines and Planes

Chapter 1: Vectors and the Geometry of Space Lesson 5 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to write equations for lines and planes in 3D space using vectors.

In one sentence: Vectors give us powerful ways to describe lines and planes in three-dimensional space.

💡 The Idea

Equations of Lines

A line in 3D can be described using a point and a direction vector.

Vector Equation of a Line

If a line passes through point P₀(x₀, y₀, z₀) and has direction vector v = ⟨a, b, c⟩:

r = r₀ + tv

Where r = ⟨x, y, z⟩ and r₀ = ⟨x₀, y₀, z₀⟩

Parametric Equations

The vector equation gives us three parametric equations:

x = x₀ + at
y = y₀ + bt
z = z₀ + ct

Where t is the parameter.

Symmetric Equations

If a, b, c are all nonzero:

(x - x₀)/a = (y - y₀)/b = (z - z₀)/c

Equations of Planes

A plane in 3D can be described using a point and a normal vector.

Vector Equation of a Plane

If a plane passes through point P₀(x₀, y₀, z₀) and has normal vector n = ⟨a, b, c⟩:

n · (r - r₀) = 0

Where r = ⟨x, y, z⟩ and r₀ = ⟨x₀, y₀, z₀⟩

Scalar Equation of a Plane

The vector equation simplifies to:

a(x - x₀) + b(y - y₀) + c(z - z₀) = 0

Or:

ax + by + cz = d

Where d = ax₀ + by₀ + cz₀

Finding the Normal Vector

The normal vector to a plane can be found using the cross product of two vectors in the plane.

Example

If you have two vectors u and v in the plane:

n = u × v

This gives a vector perpendicular to both u and v, hence perpendicular to the plane.

Distance from a Point to a Plane

The distance from point P(x₁, y₁, z₁) to the plane ax + by + cz = d is:

D = |ax₁ + by₁ + cz₁ - d| / √(a² + b² + c²)

Why This Matters

Understanding lines and planes in 3D is essential for computer graphics, robotics, physics, and engineering. These equations form the foundation for more complex geometry.

📝 Example

Example 1: Finding the Equation of a Line

Problem: Find the equation of the line through (1, 2, 3) with direction vector ⟨2, -1, 4⟩

Step 1: Write the vector equation

r = ⟨1, 2, 3⟩ + t⟨2, -1, 4⟩

Step 2: Write the parametric equations

x = 1 + 2t
y = 2 - t
z = 3 + 4t

Step 3: Write the symmetric equations

(x - 1)/2 = (y - 2)/(-1) = (z - 3)/4

Example 2: Finding the Equation of a Plane

Problem: Find the equation of the plane through (1, 2, 3) with normal vector ⟨4, 5, 6⟩

Step 1: Use the scalar equation

a(x - x₀) + b(y - y₀) + c(z - z₀) = 0

Step 2: Substitute

4(x - 1) + 5(y - 2) + 6(z - 3) = 0

Step 3: Simplify

4x - 4 + 5y - 10 + 6z - 18 = 0
4x + 5y + 6z = 32

Example 3: Finding the Distance from a Point to a Plane

Problem: Find the distance from (1, 2, 3) to the plane 2x + 3y + 4z = 10

Step 1: Use the distance formula

D = |ax₁ + by₁ + cz₁ - d| / √(a² + b² + c²)

Step 2: Substitute

D = |2(1) + 3(2) + 4(3) - 10| / √(2² + 3² + 4²)
= |2 + 6 + 12 - 10| / √(4 + 9 + 16)
= |10| / √29
= 10/√29 ≈ 1.86

📋 How to Solve Most Questions

Step-by-Step: Finding the Equation of a Line

  1. Identify a point — Find P₀(x₀, y₀, z₀) on the line
  2. Find the direction vector — Determine v = ⟨a, b, c⟩
  3. Write the vector equation — r = r₀ + tv
  4. Write parametric equations — x = x₀ + at, etc.
  5. Write symmetric equations — (x - x₀)/a = (y - y₀)/b = (z - z₀)/c

Step-by-Step: Finding the Equation of a Plane

  1. Identify a point — Find P₀(x₀, y₀, z₀) on the plane
  2. Find the normal vector — Determine n = ⟨a, b, c⟩
  3. Use the scalar equation — a(x - x₀) + b(y - y₀) + c(z - z₀) = 0
  4. Simplify — Write in the form ax + by + cz = d

The Dot Product

Chapter 1: Vectors and the Geometry of Space Lesson 3 of 6 ⏱️ 30-40 minutes

🎯 Goal

Learn how to compute the dot product of two vectors and understand its geometric meaning.

In one sentence: The dot product is a way to multiply two vectors that gives a scalar (number) and measures how much they point in the same direction.

💡 The Idea

What is the Dot Product?

The dot product (also called the scalar product) of two vectors is a number, not a vector.

If a = ⟨a₁, a₂, a₃⟩ and b = ⟨b₁, b₂, b₃⟩, then:
a · b = a₁b₁ + a₂b₂ + a₃b₃

Example

If a = ⟨1, 2, 3⟩ and b = ⟨4, 5, 6⟩:

a · b = (1)(4) + (2)(5) + (3)(6) = 4 + 10 + 18 = 32

Geometric Interpretation

The dot product has a beautiful geometric meaning:

a · b = |a| |b| cos(θ)

Where θ is the angle between the two vectors.

What This Means

  • If θ = 0° (vectors point the same way): a · b = |a| |b| (maximum)
  • If θ = 90° (vectors are perpendicular): a · b = 0
  • If θ = 180° (vectors point opposite ways): a · b = -|a| |b| (minimum)

Finding the Angle Between Vectors

We can rearrange the formula to find the angle:

cos(θ) = (a · b) / (|a| |b|)

Example: Finding the Angle

Find the angle between a = ⟨1, 0⟩ and b = ⟨1, 1⟩

a · b = (1)(1) + (0)(1) = 1
|a| = √(1² + 0²) = 1
|b| = √(1² + 1²) = √2
cos(θ) = 1 / (1 · √2) = 1/√2
θ = 45°

Properties of the Dot Product

  • Commutative: a · b = b · a
  • Distributive: a · (b + c) = a · b + a · c
  • Scalar multiplication: (ca) · b = c(a · b)
  • Perpendicular test: a · b = 0 if and only if a ⊥ b

Why This Matters

The dot product is essential in physics (work, flux), computer graphics (lighting, projections), and machine learning (similarity measures). It's one of the most useful operations in mathematics.

📝 Example

Example 1: Computing the Dot Product

Problem: Find a · b for a = ⟨2, -1, 3⟩ and b = ⟨4, 0, -2⟩

Step 1: Multiply corresponding components

a · b = (2)(4) + (-1)(0) + (3)(-2)

Step 2: Simplify

a · b = 8 + 0 - 6 = 2

Step 3: Interpret

The dot product is 2, which is positive, so the angle between them is less than 90°.

Example 2: Finding the Angle

Problem: Find the angle between a = ⟨1, 1, 0⟩ and b = ⟨0, 1, 1⟩

Step 1: Compute the dot product

a · b = (1)(0) + (1)(1) + (0)(1) = 1

Step 2: Find the magnitudes

|a| = √(1² + 1² + 0²) = √2
|b| = √(0² + 1² + 1²) = √2

Step 3: Find cos(θ)

cos(θ) = 1 / (√2 · √2) = 1/2

Step 4: Find θ

θ = 60°

Example 3: Checking Perpendicularity

Problem: Are a = ⟨1, 2, 3⟩ and b = ⟨2, -1, 0⟩ perpendicular?

Step 1: Compute the dot product

a · b = (1)(2) + (2)(-1) + (3)(0) = 2 - 2 + 0 = 0

Step 2: Interpret

Since a · b = 0, the vectors are perpendicular (orthogonal).

📋 How to Solve Most Questions

Step-by-Step: Computing the Dot Product

  1. Write the vectors — Make sure they have the same number of components
  2. Multiply corresponding components — a₁b₁, a₂b₂, a₃b₃
  3. Add the products — Sum all the products
  4. Simplify — Get your final answer

Step-by-Step: Finding the Angle Between Vectors

  1. Compute a · b — Use the dot product formula
  2. Find |a| — Compute the magnitude of a
  3. Find |b| — Compute the magnitude of b
  4. Compute cos(θ) — Use cos(θ) = (a · b) / (|a| |b|)
  5. Find θ — Take the inverse cosine

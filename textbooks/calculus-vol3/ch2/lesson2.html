Derivatives and Integrals of Vector Functions

Chapter 2: Vector Functions Lesson 2 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to differentiate and integrate vector functions component by component.

In one sentence: You can differentiate and integrate vector functions by differentiating or integrating each component separately.

💡 The Idea

Derivatives of Vector Functions

If r(t) = ⟨f(t), g(t), h(t)⟩, then the derivative is:

r'(t) = ⟨f'(t), g'(t), h'(t)⟩

You simply differentiate each component separately!

Example

If r(t) = ⟨t², sin(t), eᵗ⟩:

r'(t) = ⟨2t, cos(t), eᵗ⟩

Integrals of Vector Functions

If r(t) = ⟨f(t), g(t), h(t)⟩, then the integral is:

∫r(t) dt = ⟨∫f(t) dt, ∫g(t) dt, ∫h(t) dt⟩

You simply integrate each component separately!

Example

If r(t) = ⟨t, cos(t), 2t⟩:

∫r(t) dt = ⟨t²/2 + C₁, sin(t) + C₂, t² + C₃⟩

Properties

Vector function derivatives follow similar rules to scalar functions:

  • Sum rule: [r(t) + s(t)]' = r'(t) + s'(t)
  • Scalar multiple: [cr(t)]' = cr'(t)
  • Product rule: [r(t) · s(t)]' = r'(t) · s(t) + r(t) · s'(t)
  • Cross product rule: [r(t) × s(t)]' = r'(t) × s(t) + r(t) × s'(t)

Why This Matters

Derivatives and integrals of vector functions are essential for understanding motion in space, calculating work, and analyzing vector fields. This is the foundation for multivariable calculus.

📝 Example

Example 1: Finding the Derivative

Problem: Find r'(t) for r(t) = ⟨t³, ln(t), 5t⟩

Step 1: Differentiate each component

d/dt[t³] = 3t²
d/dt[ln(t)] = 1/t
d/dt[5t] = 5

Step 2: Write the derivative

r'(t) = ⟨3t², 1/t, 5⟩

Example 2: Finding the Integral

Problem: Find ∫r(t) dt for r(t) = ⟨2t, sin(t), 1⟩

Step 1: Integrate each component

∫2t dt = t² + C₁
∫sin(t) dt = -cos(t) + C₂
∫1 dt = t + C₃

Step 2: Write the integral

∫r(t) dt = ⟨t² + C₁, -cos(t) + C₂, t + C₃⟩

Example 3: Finding Position from Velocity

Problem: A particle has velocity v(t) = ⟨2t, 3t², 1⟩ and starts at r(0) = ⟨1, 2, 3⟩. Find r(t).

Step 1: Integrate velocity to get position

r(t) = ∫v(t) dt = ⟨t² + C₁, t³ + C₂, t + C₃⟩

Step 2: Use initial condition

r(0) = ⟨0 + C₁, 0 + C₂, 0 + C₃⟩ = ⟨1, 2, 3⟩
C₁ = 1, C₂ = 2, C₃ = 3

Step 3: Write the final answer

r(t) = ⟨t² + 1, t³ + 2, t + 3⟩

📋 How to Solve Most Questions

Step-by-Step: Differentiating Vector Functions

  1. Write the vector function — r(t) = ⟨f(t), g(t), h(t)⟩
  2. Differentiate each component — Find f'(t), g'(t), h'(t)
  3. Write the derivative — r'(t) = ⟨f'(t), g'(t), h'(t)⟩
  4. Simplify — Combine like terms if possible

Step-by-Step: Integrating Vector Functions

  1. Write the vector function — r(t) = ⟨f(t), g(t), h(t)⟩
  2. Integrate each component — Find ∫f(t) dt, ∫g(t) dt, ∫h(t) dt
  3. Add constants — Include C₁, C₂, C₃ for each component
  4. Write the integral — ∫r(t) dt = ⟨∫f(t) dt, ∫g(t) dt, ∫h(t) dt⟩

Derivatives and Integrals of Vector Functions

Chapter 2: Vector Functions Lesson 2 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to differentiate and integrate vector functions component by component.

In one sentence: You can differentiate and integrate vector functions by differentiating or integrating each component separately.

💡 The Idea

Derivatives of Vector Functions

If r(t) = ⟨f(t), g(t), h(t)⟩, then the derivative is:

r'(t) = ⟨f'(t), g'(t), h'(t)⟩

You simply differentiate each component separately!

Integrals of Vector Functions

If r(t) = ⟨f(t), g(t), h(t)⟩, then the integral is:

∫r(t) dt = ⟨∫f(t) dt, ∫g(t) dt, ∫h(t) dt⟩

You simply integrate each component separately!

Properties

  • Linearity: d/dt[u(t) + v(t)] = u'(t) + v'(t)
  • Scalar multiplication: d/dt[cu(t)] = cu'(t)
  • Product rule: d/dt[u(t) · v(t)] = u'(t) · v(t) + u(t) · v'(t)
  • Cross product rule: d/dt[u(t) × v(t)] = u'(t) × v(t) + u(t) × v'(t)

Why This Matters

Derivatives and integrals of vector functions are essential for understanding motion in space, calculating work, and analyzing vector fields.

📝 Example

Example 1: Differentiating a Vector Function

Problem: Find r'(t) for r(t) = ⟨t², sin(t), eᵗ⟩

Step 1: Differentiate each component

d/dt[t²] = 2t
d/dt[sin(t)] = cos(t)
d/dt[eᵗ] = eᵗ

Step 2: Write the derivative

r'(t) = ⟨2t, cos(t), eᵗ⟩

Example 2: Integrating a Vector Function

Problem: Find ∫r(t) dt for r(t) = ⟨2t, cos(t), 1⟩

Step 1: Integrate each component

∫2t dt = t² + C₁
∫cos(t) dt = sin(t) + C₂
∫1 dt = t + C₃

Step 2: Write the integral

∫r(t) dt = ⟨t² + C₁, sin(t) + C₂, t + C₃⟩

Example 3: Finding Velocity from Position

Problem: A particle has position r(t) = ⟨t³, 2t, t²⟩. Find the velocity at t = 1.

Step 1: Find the velocity function

v(t) = r'(t) = ⟨3t², 2, 2t⟩

Step 2: Evaluate at t = 1

v(1) = ⟨3(1)², 2, 2(1)⟩ = ⟨3, 2, 2⟩

📋 How to Solve Most Questions

Step-by-Step: Differentiating Vector Functions

  1. Write the vector function — r(t) = ⟨f(t), g(t), h(t)⟩
  2. Differentiate each component — Find f'(t), g'(t), h'(t)
  3. Write the derivative — r'(t) = ⟨f'(t), g'(t), h'(t)⟩
  4. Evaluate if needed — Plug in specific values of t

Step-by-Step: Integrating Vector Functions

  1. Write the vector function — r(t) = ⟨f(t), g(t), h(t)⟩
  2. Integrate each component — Find ∫f(t) dt, ∫g(t) dt, ∫h(t) dt
  3. Add constants — Include C₁, C₂, C₃ for each component
  4. Write the integral — ∫r(t) dt = ⟨∫f(t) dt, ∫g(t) dt, ∫h(t) dt⟩

The Chain Rule

Chapter 2: Vector Functions Lesson 6 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to differentiate composite functions of multiple variables.

In one sentence: The chain rule for partial derivatives lets you find how a function changes when its variables are themselves functions of other variables.

💡 The Idea

The Chain Rule for One Variable

Remember the chain rule from single-variable calculus:

d/dx[f(g(x))] = f'(g(x)) · g'(x)

The Chain Rule for Two Variables

If z = f(x, y) where x = g(t) and y = h(t), then:

dz/dt = (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt)

We multiply partial derivatives by ordinary derivatives and add them up.

The Chain Rule for Partial Derivatives

If z = f(x, y) where x = g(s, t) and y = h(s, t), then:

∂z/∂s = (∂f/∂x)(∂x/∂s) + (∂f/∂y)(∂y/∂s)
∂z/∂t = (∂f/∂x)(∂x/∂t) + (∂f/∂y)(∂y/∂t)

Tree Diagrams

A tree diagram helps visualize the chain rule:

Example

For z = f(x, y) where x = g(t) and y = h(t):

z
/ \
x y
| |
t t

Each path from z to t gives a term in dz/dt.

Why This Matters

The chain rule is essential for implicit differentiation, related rates, and understanding how changes propagate through complex systems.

📝 Example

Example 1: Chain Rule with One Parameter

Problem: Find dz/dt for z = x²y where x = cos(t) and y = sin(t)

Step 1: Find partial derivatives

∂z/∂x = 2xy
∂z/∂y = x²

Step 2: Find ordinary derivatives

dx/dt = -sin(t)
dy/dt = cos(t)

Step 3: Apply the chain rule

dz/dt = (2xy)(-sin(t)) + (x²)(cos(t))
= 2cos(t)sin(t)(-sin(t)) + cos²(t)cos(t)
= -2cos(t)sin²(t) + cos³(t)

Example 2: Chain Rule with Two Parameters

Problem: Find ∂z/∂s and ∂z/∂t for z = x² + y² where x = s + t and y = s - t

Step 1: Find partial derivatives

∂z/∂x = 2x
∂z/∂y = 2y
∂x/∂s = 1, ∂x/∂t = 1
∂y/∂s = 1, ∂y/∂t = -1

Step 2: Apply the chain rule

∂z/∂s = (2x)(1) + (2y)(1) = 2x + 2y
∂z/∂t = (2x)(1) + (2y)(-1) = 2x - 2y

Step 3: Substitute

∂z/∂s = 2(s+t) + 2(s-t) = 4s
∂z/∂t = 2(s+t) - 2(s-t) = 4t

Example 3: Implicit Differentiation

Problem: Find dy/dx if x² + y² = 25

Step 1: Differentiate both sides

d/dx[x² + y²] = d/dx[25]
2x + 2y(dy/dx) = 0

Step 2: Solve for dy/dx

2y(dy/dx) = -2x
dy/dx = -x/y

📋 How to Solve Most Questions

Step-by-Step: Using the Chain Rule

  1. Draw a tree diagram — Visualize the dependencies
  2. Find partial derivatives — Compute ∂f/∂x, ∂f/∂y, etc.
  3. Find derivatives of inner functions — Compute dx/dt, dy/dt, etc.
  4. Multiply along paths — For each path from outer to inner
  5. Add the products — Sum all the terms

Step-by-Step: Implicit Differentiation

  1. Write the equation — F(x, y) = 0
  2. Differentiate both sides — Use the chain rule
  3. Solve for dy/dx — Isolate the derivative
  4. Simplify — Express in simplest form

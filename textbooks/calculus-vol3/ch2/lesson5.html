Partial Derivatives

Chapter 2: Vector Functions Lesson 5 of 7 ⏱️ 40-50 minutes

🎯 Goal

Learn how to differentiate functions of multiple variables with respect to one variable at a time.

In one sentence: A partial derivative measures how a function changes when you vary one variable while holding all others constant.

💡 The Idea

What is a Partial Derivative?

For a function f(x, y), the partial derivative with respect to x is:

∂f/∂x = lim(h→0) [f(x+h, y) - f(x, y)] / h

We treat y as a constant and differentiate with respect to x.

Notation

Partial derivatives can be written as:

∂f/∂x = fₓ = ∂z/∂x

All three notations mean the same thing.

Higher-Order Partial Derivatives

You can take partial derivatives of partial derivatives:

∂²f/∂x² = fₓₓ
∂²f/∂y² = fᵧᵧ
∂²f/∂x∂y = fₓᵧ
∂²f/∂y∂x = fᵧₓ

For most functions, fₓᵧ = fᵧₓ (Clairaut's theorem).

Why This Matters

Partial derivatives are essential for understanding how functions of multiple variables change. They're used in optimization, physics, economics, and machine learning.

📝 Example

Example 1: Finding Partial Derivatives

Problem: Find ∂f/∂x and ∂f/∂y for f(x, y) = x²y + 3xy²

Step 1: Find ∂f/∂x

Treat y as a constant:

∂f/∂x = 2xy + 3y²

Step 2: Find ∂f/∂y

Treat x as a constant:

∂f/∂y = x² + 6xy

Example 2: Evaluating Partial Derivatives

Problem: Find fₓ(1, 2) for f(x, y) = x³y²

Step 1: Find fₓ

fₓ = 3x²y²

Step 2: Evaluate at (1, 2)

fₓ(1, 2) = 3(1)²(2)² = 3 · 1 · 4 = 12

Example 3: Finding Second Partial Derivatives

Problem: Find all second partial derivatives of f(x, y) = x²y + y³

Step 1: Find first partials

fₓ = 2xy
fᵧ = x² + 3y²

Step 2: Find second partials

fₓₓ = 2y
fᵧᵧ = 6y
fₓᵧ = 2x
fᵧₓ = 2x

Step 3: Verify Clairaut's theorem

fₓᵧ = fᵧₓ = 2x ✓

📋 How to Solve Most Questions

Step-by-Step: Finding Partial Derivatives

  1. Identify the variable — Which variable are you differentiating with respect to?
  2. Treat other variables as constants — They don't change
  3. Differentiate normally — Use standard differentiation rules
  4. Simplify — Combine like terms
  5. Evaluate if needed — Plug in specific values

Step-by-Step: Finding Second Partial Derivatives

  1. Find first partial derivatives — fₓ and fᵧ
  2. Differentiate again — Take partial derivatives of fₓ and fᵧ
  3. Find all four second partials — fₓₓ, fᵧᵧ, fₓᵧ, fᵧₓ
  4. Check Clairaut's theorem — Verify fₓᵧ = fᵧₓ

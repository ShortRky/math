Arc Length and Curvature

Chapter 2: Vector Functions Lesson 3 of 7 ⏱️ 40-50 minutes

🎯 Goal

Learn how to calculate the length of a curve and measure how sharply it bends.

In one sentence: Arc length measures how long a curve is, and curvature measures how quickly it changes direction.

💡 The Idea

Arc Length

The arc length of a curve from t = a to t = b is:

L = ∫ₐᵇ |r'(t)| dt

Where |r'(t)| is the magnitude of the derivative (the speed).

Curvature

Curvature measures how quickly a curve changes direction. The formula is:

κ = |r'(t) × r''(t)| / |r'(t)|³

Or equivalently:

κ = |T'(t)| / |r'(t)|

Where T(t) is the unit tangent vector.

Unit Tangent Vector

The unit tangent vector points in the direction of motion:

T(t) = r'(t) / |r'(t)|

Radius of Curvature

The radius of curvature is the reciprocal of curvature:

R = 1/κ

A smaller radius means a sharper bend.

Why This Matters

Arc length and curvature are essential for understanding motion, designing roads and roller coasters, and analyzing the geometry of curves.

📝 Example

Example 1: Finding Arc Length

Problem: Find the arc length of r(t) = ⟨cos(t), sin(t), t⟩ from t = 0 to t = 2π

Step 1: Find r'(t)

r'(t) = ⟨-sin(t), cos(t), 1⟩

Step 2: Find |r'(t)|

|r'(t)| = √[(-sin(t))² + (cos(t))² + 1²]
= √[sin²(t) + cos²(t) + 1]
= √[1 + 1] = √2

Step 3: Integrate

L = ∫₀²π √2 dt = √2 · 2π = 2π√2

Example 2: Finding Curvature

Problem: Find the curvature of r(t) = ⟨t, t², 0⟩ at t = 0

Step 1: Find r'(t) and r''(t)

r'(t) = ⟨1, 2t, 0⟩
r''(t) = ⟨0, 2, 0⟩

Step 2: Find r'(t) × r''(t)

r'(t) × r''(t) = ⟨1, 2t, 0⟩ × ⟨0, 2, 0⟩
= ⟨0, 0, 2⟩

Step 3: Find |r'(t) × r''(t)|

|r'(t) × r''(t)| = 2

Step 4: Find |r'(t)|³

|r'(t)| = √(1 + 4t²)
|r'(t)|³ = (1 + 4t²)^(3/2)

Step 5: Find κ at t = 0

κ = 2 / (1 + 0)^(3/2) = 2

Example 3: Finding the Unit Tangent Vector

Problem: Find T(t) for r(t) = ⟨t, t²⟩

Step 1: Find r'(t)

r'(t) = ⟨1, 2t⟩

Step 2: Find |r'(t)|

|r'(t)| = √(1 + 4t²)

Step 3: Find T(t)

T(t) = ⟨1, 2t⟩ / √(1 + 4t²)

📋 How to Solve Most Questions

Step-by-Step: Finding Arc Length

  1. Find r'(t) — Differentiate the vector function
  2. Find |r'(t)| — Compute the magnitude
  3. Set up the integral — L = ∫ₐᵇ |r'(t)| dt
  4. Evaluate — Compute the definite integral

Step-by-Step: Finding Curvature

  1. Find r'(t) and r''(t) — Compute first and second derivatives
  2. Find r'(t) × r''(t) — Compute the cross product
  3. Find |r'(t) × r''(t)| — Compute the magnitude
  4. Find |r'(t)|³ — Cube the magnitude of r'(t)
  5. Compute κ — Divide: κ = |r'(t) × r''(t)| / |r'(t)|³

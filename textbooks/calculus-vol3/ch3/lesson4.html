Cylindrical Coordinates

Chapter 3: Multiple Integrals Lesson 4 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn how to use cylindrical coordinates to evaluate triple integrals over regions with cylindrical symmetry.

In one sentence: Cylindrical coordinates combine polar coordinates with the z-axis, making it easier to integrate over cylindrical regions.

💡 The Idea

Cylindrical Coordinates

Cylindrical coordinates use r, θ, and z:

x = r cos(θ)
y = r sin(θ)
z = z

Where r is the distance from the z-axis, θ is the angle, and z is the height.

Volume Element

The volume element dV in cylindrical coordinates is:

dV = r dz dr dθ

The factor r comes from the Jacobian of the transformation from Cartesian to cylindrical.

Triple Integral in Cylindrical

The triple integral of f(x, y, z) over a region E becomes:

∭ₑ f(x, y, z) dV = ∭ₑ f(r cos(θ), r sin(θ), z) r dz dr dθ

When to Use Cylindrical

  • Cylindrical regions — cylinders, cones, paraboloids
  • Integrands with x² + y² — becomes r²
  • Symmetry about z-axis — simplifies bounds

Why This Matters

Cylindrical coordinates are essential for solving integrals over cylindrical regions that would be very difficult in Cartesian coordinates. They're widely used in physics and engineering for problems with cylindrical symmetry.

📝 Example

Example 1: Volume of a Cylinder

Problem: Find the volume of the cylinder x² + y² ≤ 4, 0 ≤ z ≤ 3

Step 1: Convert to cylindrical

x² + y² = r²
Bounds: 0 ≤ r ≤ 2, 0 ≤ θ ≤ 2π, 0 ≤ z ≤ 3

Step 2: Set up the integral

V = ∫₀²π ∫₀² ∫₀³ r dz dr dθ

Step 3: Integrate with respect to z

∫₀³ r dz = rz |₀³ = 3r

Step 4: Integrate with respect to r

∫₀² 3r dr = [3r²/2]₀² = 6

Step 5: Integrate with respect to θ

∫₀²π 6 dθ = 6θ |₀²π = 12π

The volume is 12π cubic units.

Example 2: Mass of a Cylinder

Problem: Find the mass of the cylinder x² + y² ≤ 1, 0 ≤ z ≤ 2 with density ρ(x, y, z) = z

Step 1: Convert to cylindrical

Bounds: 0 ≤ r ≤ 1, 0 ≤ θ ≤ 2π, 0 ≤ z ≤ 2
Density: ρ = z

Step 2: Set up the integral

M = ∫₀²π ∫₀¹ ∫₀² z · r dz dr dθ

Step 3: Integrate with respect to z

∫₀² zr dz = r[z²/2]₀² = 2r

Step 4: Integrate with respect to r

∫₀¹ 2r dr = [r²]₀¹ = 1

Step 5: Integrate with respect to θ

∫₀²π 1 dθ = θ |₀²π = 2π

The mass is 2π units.

Example 3: Integral over a Cone

Problem: Evaluate ∭ₑ z dV where E is the cone z = √(x² + y²), 0 ≤ z ≤ 2

Step 1: Convert to cylindrical

z = √(x² + y²) = r
Bounds: 0 ≤ r ≤ 2, 0 ≤ θ ≤ 2π, r ≤ z ≤ 2

Step 2: Set up the integral

∫₀²π ∫₀² ∫ʳ² z · r dz dr dθ

Step 3: Integrate with respect to z

∫ʳ² zr dz = r[z²/2]ʳ² = r(2 - r²/2) = 2r - r³/2

Step 4: Integrate with respect to r

∫₀² (2r - r³/2) dr = [r² - r⁴/8]₀² = 4 - 2 = 2

Step 5: Integrate with respect to θ

∫₀²π 2 dθ = 2θ |₀²π = 4π

📋 How to Solve Most Questions

Step-by-Step: Using Cylindrical Coordinates

  1. Identify the region — Is it cylindrical or symmetric about z-axis?
  2. Convert the integrand — Replace x² + y² with r², x with r cos(θ), y with r sin(θ)
  3. Include the Jacobian — Multiply by r
  4. Find the bounds — Determine r, θ, and z limits
  5. Evaluate — Integrate with respect to z first, then r, then θ

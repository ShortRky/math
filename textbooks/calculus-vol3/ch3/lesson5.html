Spherical Coordinates

Chapter 3: Multiple Integrals Lesson 5 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to use spherical coordinates to evaluate triple integrals over regions with spherical symmetry.

In one sentence: Spherical coordinates use radius and two angles to describe points in 3D, making it easier to integrate over spherical regions.

💡 The Idea

Spherical Coordinates

Spherical coordinates use ρ, θ, and φ:

x = ρ sin(φ) cos(θ)
y = ρ sin(φ) sin(θ)
z = ρ cos(φ)

Where:

  • ρ is the distance from the origin
  • θ is the angle in the xy-plane (same as cylindrical)
  • φ is the angle from the positive z-axis

Volume Element

The volume element dV in spherical coordinates is:

dV = ρ² sin(φ) dρ dφ dθ

The factor ρ² sin(φ) comes from the Jacobian of the transformation.

Triple Integral in Spherical

The triple integral of f(x, y, z) over a region E becomes:

∭ₑ f(x, y, z) dV = ∭ₑ f(ρ sin(φ) cos(θ), ρ sin(φ) sin(θ), ρ cos(φ)) ρ² sin(φ) dρ dφ dθ

When to Use Spherical

  • Spherical regions — balls, spheres, hemispheres
  • Integrands with x² + y² + z² — becomes ρ²
  • Symmetry about origin — simplifies bounds

Why This Matters

Spherical coordinates are essential for solving integrals over spherical regions that would be very difficult in Cartesian coordinates. They're widely used in physics for gravitational and electromagnetic fields.

📝 Example

Example 1: Volume of a Sphere

Problem: Find the volume of the sphere x² + y² + z² ≤ 4

Step 1: Convert to spherical

x² + y² + z² = ρ²
Bounds: 0 ≤ ρ ≤ 2, 0 ≤ θ ≤ 2π, 0 ≤ φ ≤ π

Step 2: Set up the integral

V = ∫₀²π ∫₀π ∫₀² ρ² sin(φ) dρ dφ dθ

Step 3: Integrate with respect to ρ

∫₀² ρ² dρ = [ρ³/3]₀² = 8/3

Step 4: Integrate with respect to φ

∫₀π (8/3) sin(φ) dφ = (8/3)[-cos(φ)]₀π = (8/3)(2) = 16/3

Step 5: Integrate with respect to θ

∫₀²π (16/3) dθ = (16/3)θ |₀²π = 32π/3

The volume is 32π/3 cubic units.

Example 2: Mass of a Sphere

Problem: Find the mass of the sphere x² + y² + z² ≤ 1 with density ρ(x, y, z) = z²

Step 1: Convert to spherical

z = ρ cos(φ)
z² = ρ² cos²(φ)
Bounds: 0 ≤ ρ ≤ 1, 0 ≤ θ ≤ 2π, 0 ≤ φ ≤ π

Step 2: Set up the integral

M = ∫₀²π ∫₀π ∫₀¹ ρ² cos²(φ) · ρ² sin(φ) dρ dφ dθ

Step 3: Integrate with respect to ρ

∫₀¹ ρ⁴ dρ = [ρ⁵/5]₀¹ = 1/5

Step 4: Integrate with respect to φ

∫₀π (1/5) cos²(φ) sin(φ) dφ = (1/5)[-cos³(φ)/3]₀π = (1/5)(2/3) = 2/15

Step 5: Integrate with respect to θ

∫₀²π (2/15) dθ = (2/15)θ |₀²π = 4π/15

The mass is 4π/15 units.

Example 3: Integral over a Hemisphere

Problem: Evaluate ∭ₑ z dV where E is the upper hemisphere x² + y² + z² ≤ 1, z ≥ 0

Step 1: Convert to spherical

z = ρ cos(φ)
Bounds: 0 ≤ ρ ≤ 1, 0 ≤ θ ≤ 2π, 0 ≤ φ ≤ π/2

Step 2: Set up the integral

∫₀²π ∫₀π/² ∫₀¹ ρ cos(φ) · ρ² sin(φ) dρ dφ dθ

Step 3: Integrate with respect to ρ

∫₀¹ ρ³ dρ = [ρ⁴/4]₀¹ = 1/4

Step 4: Integrate with respect to φ

∫₀π/² (1/4) cos(φ) sin(φ) dφ = (1/4)[sin²(φ)/2]₀π/² = 1/8

Step 5: Integrate with respect to θ

∫₀²π (1/8) dθ = (1/8)θ |₀²π = π/4

📋 How to Solve Most Questions

Step-by-Step: Using Spherical Coordinates

  1. Identify the region — Is it spherical or symmetric about origin?
  2. Convert the integrand — Replace x² + y² + z² with ρ², z with ρ cos(φ)
  3. Include the Jacobian — Multiply by ρ² sin(φ)
  4. Find the bounds — Determine ρ, θ, and φ limits
  5. Evaluate — Integrate with respect to ρ first, then φ, then θ

Triple Integrals

Chapter 3: Multiple Integrals Lesson 3 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to evaluate triple integrals to find volumes and other quantities in three-dimensional regions.

In one sentence: Triple integrals extend double integrals to three dimensions, allowing us to integrate over volumes.

💡 The Idea

What is a Triple Integral?

A triple integral integrates a function f(x, y, z) over a three-dimensional region E:

∭ₑ f(x, y, z) dV

Where dV = dx dy dz (or any order of differentials).

Evaluating Triple Integrals

Triple integrals are evaluated as iterated integrals, from the inside out:

∭ₑ f(x, y, z) dV = ∫∫∫ f(x, y, z) dz dy dx

You can choose the order of integration based on the region's shape.

Finding Bounds

To set up a triple integral:

  • Sketch the region — Visualize the 3D shape
  • Find outer bounds — Project onto a coordinate plane
  • Find middle bounds — For fixed outer variable
  • Find inner bounds — For fixed outer two variables

Volume

The volume of a region E is:

V = ∭ₑ dV = ∭ₑ 1 dV

Why This Matters

Triple integrals are essential for calculating volumes, masses, centers of mass, and moments of inertia in three dimensions. They're fundamental in physics and engineering.

📝 Example

Example 1: Volume of a Box

Problem: Find the volume of the box 0 ≤ x ≤ 2, 0 ≤ y ≤ 3, 0 ≤ z ≤ 1

Step 1: Set up the integral

V = ∫₀² ∫₀³ ∫₀¹ dz dy dx

Step 2: Integrate with respect to z

∫₀¹ dz = z |₀¹ = 1

Step 3: Integrate with respect to y

∫₀³ 1 dy = y |₀³ = 3

Step 4: Integrate with respect to x

∫₀² 3 dx = 3x |₀² = 6

The volume is 6 cubic units.

Example 2: Mass of a Region

Problem: Find the mass of the region 0 ≤ x ≤ 1, 0 ≤ y ≤ 1, 0 ≤ z ≤ 1 with density ρ(x, y, z) = x + y + z

Step 1: Set up the integral

M = ∫₀¹ ∫₀¹ ∫₀¹ (x + y + z) dz dy dx

Step 2: Integrate with respect to z

∫₀¹ (x + y + z) dz = [xz + yz + z²/2]₀¹ = x + y + 1/2

Step 3: Integrate with respect to y

∫₀¹ (x + y + 1/2) dy = [xy + y²/2 + y/2]₀¹ = x + 1/2 + 1/2 = x + 1

Step 4: Integrate with respect to x

∫₀¹ (x + 1) dx = [x²/2 + x]₀¹ = 1/2 + 1 = 3/2

The mass is 3/2 units.

Example 3: Changing Order of Integration

Problem: Evaluate ∫₀¹ ∫₀¹ ∫₀¹ xyz dz dy dx

Step 1: Integrate with respect to z

∫₀¹ xyz dz = [xyz²/2]₀¹ = xy/2

Step 2: Integrate with respect to y

∫₀¹ (xy/2) dy = [xy²/4]₀¹ = x/4

Step 3: Integrate with respect to x

∫₀¹ (x/4) dx = [x²/8]₀¹ = 1/8

📋 How to Solve Most Questions

Step-by-Step: Evaluating Triple Integrals

  1. Sketch the region — Visualize the 3D shape
  2. Determine the order — Choose dz dy dx or another order
  3. Find the bounds — For each variable, from inside to outside
  4. Integrate inner — With respect to the innermost variable
  5. Integrate middle — With respect to the middle variable
  6. Integrate outer — With respect to the outermost variable

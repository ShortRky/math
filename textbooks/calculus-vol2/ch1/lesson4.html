Partial Fractions

Chapter 1: Integration Techniques Lesson 4 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn how to decompose rational functions into simpler fractions that are easier to integrate.

In one sentence: Partial fractions break down complex rational functions into sums of simpler fractions that we can integrate.

💡 The Idea

What Are Partial Fractions?

Partial fractions is a technique where we rewrite a complicated rational function as a sum of simpler fractions.

Example

Instead of integrating 1/(x² - 1) directly, we write:

1/(x² - 1) = 1/[(x-1)(x+1)] = A/(x-1) + B/(x+1)

Then we find A and B, and integrate each term separately.

When to Use Partial Fractions

Use partial fractions when:

  • You have a rational function (polynomial divided by polynomial)
  • The degree of the numerator is less than the degree of the denominator
  • You can factor the denominator

Types of Partial Fractions

Distinct Linear Factors

For (x-a)(x-b) in the denominator:

P(x)/[(x-a)(x-b)] = A/(x-a) + B/(x-b)

Repeated Linear Factors

For (x-a)² in the denominator:

P(x)/(x-a)² = A/(x-a) + B/(x-a)²

Irreducible Quadratic Factors

For (x² + bx + c) in the denominator:

P(x)/(x² + bx + c) = (Ax + B)/(x² + bx + c)

Why This Matters

Partial fractions transform difficult integrals into simple ones. This technique is essential for integrating rational functions, which appear frequently in calculus and its applications.

📝 Example

Example 1: Distinct Linear Factors

Problem: Integrate ∫1/(x² - 1) dx

Step 1: Factor the denominator

x² - 1 = (x-1)(x+1)

Step 2: Set up partial fractions

1/[(x-1)(x+1)] = A/(x-1) + B/(x+1)

Step 3: Solve for A and B

Multiply both sides by (x-1)(x+1):

1 = A(x+1) + B(x-1)

Let x = 1: 1 = A(2) → A = 1/2

Let x = -1: 1 = B(-2) → B = -1/2

Step 4: Integrate

∫1/(x² - 1) dx = ∫[1/2/(x-1) - 1/2/(x+1)] dx
= (1/2)ln|x-1| - (1/2)ln|x+1| + C
= (1/2)ln|(x-1)/(x+1)| + C

Example 2: Repeated Linear Factors

Problem: Integrate ∫1/(x² + 2x + 1) dx

Step 1: Factor the denominator

x² + 2x + 1 = (x+1)²

Step 2: Set up partial fractions

1/(x+1)² = A/(x+1) + B/(x+1)²

Step 3: Solve for A and B

Multiply both sides by (x+1)²:

1 = A(x+1) + B

Let x = -1: 1 = B

Let x = 0: 1 = A(1) + 1 → A = 0

Step 4: Integrate

∫1/(x+1)² dx = ∫1/(x+1)² dx = -1/(x+1) + C

📋 How to Solve Most Questions

Step-by-Step: Partial Fractions

  1. Factor the denominator — Find all factors
  2. Set up partial fractions — Write the decomposition
  3. Solve for constants — Use substitution or equating coefficients
  4. Integrate each term — Use basic integration rules
  5. Simplify — Combine logarithms if possible

Step-by-Step: Solving for Constants

  1. Multiply both sides — Clear the denominators
  2. Choose convenient values — Set x to make terms zero
  3. Solve the system — Find all constants
  4. Verify — Check your values work

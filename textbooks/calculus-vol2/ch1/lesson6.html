Applications of Integration

Chapter 1: Integration Techniques Lesson 6 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to use integration to solve real-world problems including areas, volumes, and physical applications.

In one sentence: Integration allows us to calculate areas, volumes, work, and other quantities that accumulate over an interval.

💡 The Idea

Area Between Curves

The area between two curves f(x) and g(x) from a to b is:

A = ∫ₐᵇ |f(x) - g(x)| dx

Example

Find the area between y = x² and y = x from 0 to 1:

A = ∫₀¹ (x - x²) dx = [x²/2 - x³/3]₀¹ = 1/2 - 1/3 = 1/6

Volumes of Revolution

When we rotate a region around an axis, we get a solid. The volume can be found using:

Disk Method

For rotation around the x-axis:

V = π ∫ₐᵇ [f(x)]² dx

Washer Method

When there's a hole in the middle:

V = π ∫ₐᵇ ([f(x)]² - [g(x)]²) dx

Shell Method

For rotation around the y-axis:

V = 2π ∫ₐᵇ x·f(x) dx

Work

Work is force times distance. For variable force:

W = ∫ₐᵇ F(x) dx

Example

Spring force: F(x) = kx

Work to stretch spring: W = ∫₀ˣ kx dx = kx²/2

Average Value

The average value of f(x) on [a, b] is:

f_avg = (1/(b-a)) ∫ₐᵇ f(x) dx

Why This Matters

Integration is one of the most powerful tools in mathematics. It allows us to solve problems in physics, engineering, economics, and many other fields by accumulating quantities over intervals.

📝 Example

Example 1: Area Between Curves

Problem: Find the area between y = x² and y = 2x

Step 1: Find intersection points

x² = 2x → x² - 2x = 0 → x(x-2) = 0 → x = 0, 2

Step 2: Determine which is on top

At x = 1: y = 1² = 1, y = 2(1) = 2

So 2x is on top.

Step 3: Set up the integral

A = ∫₀² (2x - x²) dx

Step 4: Evaluate

A = [x² - x³/3]₀² = (4 - 8/3) - 0 = 4/3

Example 2: Volume of Revolution

Problem: Find the volume when y = √x is rotated around the x-axis from 0 to 4

Step 1: Set up the disk method

V = π ∫₀⁴ [√x]² dx = π ∫₀⁴ x dx

Step 2: Evaluate

V = π [x²/2]₀⁴ = π (16/2 - 0) = 8π

Example 3: Work

Problem: Find the work to stretch a spring 0.1 m if k = 500 N/m

Step 1: Use the work formula

W = ∫₀^0.1 500x dx

Step 2: Evaluate

W = [250x²]₀^0.1 = 250(0.01) - 0 = 2.5 J

📋 How to Solve Most Questions

Step-by-Step: Finding Area Between Curves

  1. Sketch the region — Draw the curves
  2. Find intersection points — Set f(x) = g(x)
  3. Determine top and bottom — Which curve is on top?
  4. Set up the integral — ∫(top - bottom) dx
  5. Evaluate — Compute the definite integral

Step-by-Step: Finding Volume of Revolution

  1. Sketch the region — Draw the area to be rotated
  2. Identify the axis — What are you rotating around?
  3. Choose the method — Disk, washer, or shell?
  4. Set up the integral — Use the appropriate formula
  5. Evaluate — Compute the definite integral

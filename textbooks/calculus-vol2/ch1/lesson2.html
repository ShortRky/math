Trigonometric Integrals

Chapter 1: Integration Techniques Lesson 2 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn techniques for integrating products of sine and cosine functions.

In one sentence: Use trigonometric identities and substitution to simplify integrals of trig functions.

💡 The Idea

Types of Trigonometric Integrals

We'll focus on integrals of the form:

  • ∫sinⁿ(x)·cosᵐ(x) dx
  • ∫tanⁿ(x)·secᵐ(x) dx
  • ∫sin(ax)·cos(bx) dx

Key Identities

These identities are essential:

sin²(x) + cos²(x) = 1
sin²(x) = (1 - cos(2x))/2
cos²(x) = (1 + cos(2x))/2
sin(x)·cos(x) = sin(2x)/2

Strategy for sinⁿ(x)·cosᵐ(x)

If n is odd: Save one sin(x), convert the rest to cos(x) using sin²(x) = 1 - cos²(x), then substitute u = cos(x)

If m is odd: Save one cos(x), convert the rest to sin(x) using cos²(x) = 1 - sin²(x), then substitute u = sin(x)

If both are even: Use power-reducing identities

Why This Matters

Trigonometric integrals appear everywhere in physics — wave motion, signal processing, and oscillations. Mastering these techniques is essential for engineering and science.

📝 Example

Example 1: ∫sin³(x)·cos(x) dx

Step 1: Identify the pattern

n = 3 (odd), m = 1 (odd)

Since n is odd, save one sin(x) and convert the rest

Step 2: Rewrite

∫sin³(x)·cos(x) dx = ∫sin²(x)·sin(x)·cos(x) dx
= ∫(1 - cos²(x))·sin(x)·cos(x) dx

Step 3: Substitute

Let u = cos(x), then du = -sin(x) dx

= ∫(1 - u²)·u·(-du)
= -∫(u - u³) du
= -[u²/2 - u⁴/4] + C
= -cos²(x)/2 + cos⁴(x)/4 + C

Example 2: ∫sin²(x) dx

Step 1: Identify the pattern

n = 2 (even), m = 0 (even)

Both are even, so use power-reducing identity

Step 2: Apply identity

∫sin²(x) dx = ∫(1 - cos(2x))/2 dx
= (1/2)∫dx - (1/2)∫cos(2x) dx
= x/2 - sin(2x)/4 + C

📋 How to Solve Most Questions

Step-by-Step: Integrating sinⁿ(x)·cosᵐ(x)

  1. Check the powers — Is n odd? Is m odd? Are both even?
  2. If n is odd: Save one sin(x), convert rest to cos(x), substitute u = cos(x)
  3. If m is odd: Save one cos(x), convert rest to sin(x), substitute u = sin(x)
  4. If both even: Use sin²(x) = (1-cos(2x))/2 and cos²(x) = (1+cos(2x))/2
  5. Simplify — Expand and integrate term by term
  6. Add +C — Don't forget the constant

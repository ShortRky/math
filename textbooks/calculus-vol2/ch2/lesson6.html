Taylor Series

Chapter 2: Sequences and Series Lesson 6 of 7 ⏱️ 40-50 minutes

🎯 Goal

Understand Taylor series as infinite polynomials that exactly represent functions, and learn how to find and use them.

In one sentence: A Taylor series is an infinite polynomial that exactly equals a function within its interval of convergence.

💡 The Idea

From Polynomials to Series

A Taylor polynomial approximates a function. What happens if we let the degree go to infinity? We get a Taylor series!

f(x) = ∑(n=0 to ∞) f⁽ⁿ⁾(a)(x-a)ⁿ/n!

Common Taylor Series

Exponential Function

eˣ = ∑(n=0 to ∞) xⁿ/n! = 1 + x + x²/2! + x³/3! + ...

Converges for all x.

Sine Function

sin(x) = ∑(n=0 to ∞) (-1)ⁿx²ⁿ⁺¹/(2n+1)!

= x - x³/3! + x⁵/5! - x⁷/7! + ...

Converges for all x.

Cosine Function

cos(x) = ∑(n=0 to ∞) (-1)ⁿx²ⁿ/(2n)!

= 1 - x²/2! + x⁴/4! - x⁶/6! + ...

Converges for all x.

Geometric Series

1/(1-x) = ∑(n=0 to ∞) xⁿ = 1 + x + x² + x³ + ...

Converges for |x| < 1.

Why Taylor Series Work

Taylor series work because they match all derivatives of the function at a point. If two functions have the same derivatives at a point, they're the same function (within the interval of convergence).

Why This Matters

Taylor series are fundamental in mathematics, physics, and engineering. They allow us to represent complex functions as infinite polynomials, making calculations possible that would otherwise be impossible.

📝 Example

Example 1: Finding a Taylor Series

Problem: Find the Taylor series for f(x) = eˣ centered at a = 0

Step 1: Find derivatives

f⁽ⁿ⁾(x) = eˣ for all n
f⁽ⁿ⁾(0) = 1 for all n

Step 2: Build the series

eˣ = ∑(n=0 to ∞) 1·xⁿ/n! = ∑(n=0 to ∞) xⁿ/n!

Step 3: Write out terms

eˣ = 1 + x + x²/2! + x³/3! + x⁴/4! + ...

Example 2: Using Taylor Series

Problem: Find the exact value of ∑(n=0 to ∞) (ln 2)ⁿ/n!

Step 1: Recognize the pattern

This looks like the Taylor series for eˣ with x = ln 2.

Step 2: Apply the formula

∑(n=0 to ∞) (ln 2)ⁿ/n! = e^(ln 2) = 2

Example 3: Deriving a New Series

Problem: Find the Taylor series for eˣ²

Step 1: Start with known series

eˣ = ∑(n=0 to ∞) xⁿ/n!

Step 2: Substitute x² for x

eˣ² = ∑(n=0 to ∞) (x²)ⁿ/n! = ∑(n=0 to ∞) x²ⁿ/n!

Step 3: Write out terms

eˣ² = 1 + x² + x⁴/2! + x⁶/3! + x⁸/4! + ...

📋 How to Solve Most Questions

Step-by-Step: Finding a Taylor Series

  1. Find all derivatives — Compute f⁽ⁿ⁾(x) for all n
  2. Evaluate at a — Find f⁽ⁿ⁾(a) for all n
  3. Build the series — Use the Taylor series formula
  4. Find the interval of convergence — Use the Ratio Test

Step-by-Step: Using Known Series

  1. Recognize the pattern — Does it match a known series?
  2. Substitute — Replace x with the appropriate expression
  3. Simplify — Write the new series
  4. Find the interval — Adjust for the substitution

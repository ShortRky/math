Applications of Series

Chapter 2: Sequences and Series Lesson 7 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to use Taylor series to approximate functions, evaluate integrals, solve differential equations, and understand error bounds.

In one sentence: Taylor series are powerful tools for approximating functions and solving problems that are otherwise difficult or impossible.

💡 The Idea

Why Use Series?

Series allow us to:

  • Approximate functions that are hard to compute
  • Evaluate integrals that don't have elementary antiderivatives
  • Solve differential equations
  • Understand the behavior of functions

Approximating Functions

We can use Taylor polynomials to approximate function values:

Example: Approximating e⁰·¹

Using the Taylor series for eˣ:

eˣ = 1 + x + x²/2! + x³/3! + x⁴/4! + ...

For x = 0.1:

e⁰·¹ ≈ 1 + 0.1 + (0.1)²/2 + (0.1)³/6 + (0.1)⁴/24
= 1 + 0.1 + 0.005 + 0.000167 + 0.00000417
≈ 1.105171

Actual value: e⁰·¹ ≈ 1.105171

Error Bounds

How accurate is our approximation? The error is bounded by the next term:

|Error| ≤ |next term| = |f⁽ⁿ⁺¹⁾(c)(x-a)ⁿ⁺¹/(n+1)!|

Where c is between a and x.

Example: Error in Approximating e⁰·¹

Using 4 terms (degree 3 polynomial):

|Error| ≤ (0.1)⁵/5! = 0.000001/120 ≈ 0.0000000083

Our approximation is accurate to at least 7 decimal places!

Evaluating Integrals

Some integrals don't have elementary antiderivatives. We can use series to evaluate them.

Example: ∫eˣ² dx

This integral doesn't have an elementary antiderivative.

Using the Taylor series for eˣ²:

eˣ² = 1 + x² + x⁴/2! + x⁶/3! + x⁸/4! + ...

Integrate term by term:

∫eˣ² dx = C + x + x³/3 + x⁵/(5·2!) + x⁷/(7·3!) + x⁹/(9·4!) + ...

Solving Differential Equations

Series can solve differential equations that are difficult to solve otherwise.

Example: y' = y, y(0) = 1

Assume y = ∑aₙxⁿ

Then y' = ∑n·aₙxⁿ⁻¹

Substituting into y' = y:

∑n·aₙxⁿ⁻¹ = ∑aₙxⁿ

Matching coefficients: aₙ₊₁ = aₙ/(n+1)

With a₀ = 1: aₙ = 1/n!

So y = ∑xⁿ/n! = eˣ

Why This Matters

Series are essential tools in mathematics, science, and engineering. They allow us to solve problems that would otherwise be impossible and provide deep insights into the behavior of functions.

📝 Example

Example 1: Approximating sin(0.5)

Step 1: Use the Taylor series for sin(x)

sin(x) = x - x³/3! + x⁵/5! - x⁷/7! + ...

Step 2: Substitute x = 0.5

sin(0.5) ≈ 0.5 - (0.5)³/6 + (0.5)⁵/120 - (0.5)⁷/5040
= 0.5 - 0.020833 + 0.000260 - 0.00000155
≈ 0.479425

Step 3: Check error

|Error| ≤ (0.5)⁹/9! ≈ 0.0000000068

Actual value: sin(0.5) ≈ 0.479426

Example 2: Evaluating ∫sin(x)/x dx

Step 1: Use the Taylor series for sin(x)

sin(x) = x - x³/3! + x⁵/5! - x⁷/7! + ...

Step 2: Divide by x

sin(x)/x = 1 - x²/3! + x⁴/5! - x⁶/7! + ...

Step 3: Integrate term by term

∫sin(x)/x dx = C + x - x³/(3·3!) + x⁵/(5·5!) - x⁷/(7·7!) + ...

Step 4: Evaluate from 0 to 1

∫₀¹ sin(x)/x dx ≈ 1 - 1/(3·6) + 1/(5·120) - 1/(7·5040)
= 1 - 0.055556 + 0.001667 - 0.0000283
≈ 0.946083

Example 3: Solving y'' + y = 0

Step 1: Assume a power series solution

y = ∑aₙxⁿ

Step 2: Find derivatives

y' = ∑n·aₙxⁿ⁻¹
y'' = ∑n(n-1)·aₙxⁿ⁻²

Step 3: Substitute into the equation

∑n(n-1)·aₙxⁿ⁻² + ∑aₙxⁿ = 0

Step 4: Match coefficients

For n ≥ 2: n(n-1)aₙ + aₙ₋₂ = 0

So: aₙ = -aₙ₋₂/[n(n-1)]

Step 5: Find the pattern

With a₀ = 1, a₁ = 0:

y = 1 - x²/2! + x⁴/4! - x⁶/6! + ... = cos(x)

📋 How to Solve Most Questions

Step-by-Step: Approximating Function Values

  1. Choose the series — Use a known Taylor series
  2. Substitute x — Plug in the value you want
  3. Add terms — Include enough terms for desired accuracy
  4. Check error — Use the next term as a bound
  5. State your answer — Include the approximation and error bound

Step-by-Step: Evaluating Integrals with Series

  1. Find the series — Expand the integrand
  2. Integrate term by term — Apply ∫ to each term
  3. Add the constant — Don't forget +C
  4. Evaluate if needed — Plug in limits
  5. Check convergence — Make sure the series converges

Step-by-Step: Solving Differential Equations

  1. Assume a series — Let y = ∑aₙxⁿ
  2. Find derivatives — Compute y', y'', etc.
  3. Substitute — Plug into the differential equation
  4. Match coefficients — Set up recurrence relations
  5. Solve for aₙ — Find the pattern
  6. Write the solution — Express as a series

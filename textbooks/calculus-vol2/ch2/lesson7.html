Applications of Series

Chapter 2: Sequences and Series Lesson 7 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to use Taylor series to approximate functions, evaluate integrals, solve differential equations, and understand error bounds.

In one sentence: Taylor series are powerful tools for approximating functions and solving problems that are otherwise difficult or impossible.

💡 The Idea

Approximating Functions

We can use Taylor polynomials to approximate function values:

Example

To approximate e⁰·¹, use:

eˣ ≈ 1 + x + x²/2! + x³/3! + ...

With x = 0.1:

e⁰·¹ ≈ 1 + 0.1 + 0.01/2 + 0.001/6 ≈ 1.10517

Error Bounds

The error in a Taylor approximation is bounded by the next term:

|Error| ≤ |f⁽ⁿ⁺¹⁾(c)|·|x-a|ⁿ⁺¹/(n+1)!

Where c is between a and x.

Evaluating Integrals

Some integrals can't be evaluated with elementary functions. Taylor series can help!

Example: ∫eˣ² dx

This integral has no elementary antiderivative.

Use the Taylor series for eˣ²:

eˣ² = 1 + x² + x⁴/2! + x⁶/3! + ...

Integrate term by term:

∫eˣ² dx = x + x³/3 + x⁵/(5·2!) + x⁷/(7·3!) + ...

Solving Differential Equations

Some differential equations can be solved using power series.

Example: y' = y

Assume y = ∑aₙxⁿ

Then y' = ∑naₙxⁿ⁻¹

Substituting: ∑naₙxⁿ⁻¹ = ∑aₙxⁿ

This gives: aₙ = aₙ₋₁/n

So: y = a₀∑xⁿ/n! = a₀eˣ

Why This Matters

Taylor series are used in calculators, computers, physics, engineering, and many other fields. They allow us to solve problems that would otherwise be impossible.

📝 Example

Example 1: Approximating with Error Bound

Problem: Approximate sin(0.5) with error less than 0.001

Step 1: Use the Taylor series for sin(x)

sin(x) = x - x³/3! + x⁵/5! - x⁷/7! + ...

Step 2: Find how many terms needed

The error is bounded by the next term:

|Error| ≤ xⁿ⁺¹/(n+1)!

For x = 0.5 and error < 0.001:

n = 3: (0.5)⁴/4! = 0.0625/24 ≈ 0.0026 > 0.001

n = 4: (0.5)⁵/5! = 0.03125/120 ≈ 0.00026 < 0.001 ✓

Step 3: Calculate

sin(0.5) ≈ 0.5 - (0.5)³/6 + (0.5)⁵/120
= 0.5 - 0.02083 + 0.00026 ≈ 0.47943

Example 2: Evaluating an Integral

Problem: Evaluate ∫₀¹ sin(x)/x dx

Step 1: Use the Taylor series for sin(x)

sin(x)/x = 1 - x²/3! + x⁴/5! - x⁶/7! + ...

Step 2: Integrate term by term

∫₀¹ sin(x)/x dx = ∫₀¹ (1 - x²/6 + x⁴/120 - ...) dx
= [x - x³/18 + x⁵/600 - ...]₀¹
= 1 - 1/18 + 1/600 - ...
≈ 0.94608

Example 3: Solving a Differential Equation

Problem: Solve y'' + y = 0 with y(0) = 1, y'(0) = 0

Step 1: Assume a power series solution

y = ∑aₙxⁿ

Step 2: Find derivatives

y' = ∑naₙxⁿ⁻¹
y'' = ∑n(n-1)aₙxⁿ⁻²

📋 How to Solve Most Questions

Step-by-Step: Approximating Values

  1. Choose the series — Use the appropriate Taylor series
  2. Substitute x — Plug in the value
  3. Calculate terms — Add enough terms for desired accuracy
  4. Check error — Use the next term as error bound

Step-by-Step: Evaluating Integrals

  1. Expand the integrand — Use Taylor series
  2. Integrate term by term — Integrate each term
  3. Evaluate — Plug in the limits
  4. Sum the series — Add enough terms

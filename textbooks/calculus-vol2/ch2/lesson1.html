Sequences

Chapter 2: Sequences and Series Lesson 1 of 7 ⏱️ 25-35 minutes

🎯 Goal

Understand what a sequence is, how to determine if it converges, and how to find its limit.

In one sentence: A sequence is an ordered list of numbers that may approach a specific value.

💡 The Idea

What is a Sequence?

A sequence is an ordered list of numbers, usually written as:

a₁, a₂, a₃, a₄, ...

Or using function notation:

aₙ = f(n), where n = 1, 2, 3, ...

Each number in the sequence is called a term.

Convergence and Divergence

A sequence converges if its terms approach a specific value as n gets larger:

lim(n→∞) aₙ = L

If the limit exists, the sequence converges to L. If the limit doesn't exist, the sequence diverges.

Common Sequences

Arithmetic Sequence

aₙ = a₁ + (n-1)d

Example: 2, 5, 8, 11, ... (d = 3)

These always diverge (unless d = 0)

Geometric Sequence

aₙ = a₁ · rⁿ⁻¹

Example: 1, 1/2, 1/4, 1/8, ... (r = 1/2)

Converges if |r| < 1, diverges if |r| ≥ 1

Why This Matters

Sequences are the building blocks of series. Understanding convergence of sequences is essential for understanding infinite sums, which appear throughout mathematics, physics, and engineering.

📝 Example

Example 1: Does aₙ = 1/n converge?

Step 1: Write out some terms

1, 1/2, 1/3, 1/4, 1/5, ...

Step 2: Find the limit

lim(n→∞) 1/n = 0

Step 3: Conclusion

The sequence converges to 0.

Example 2: Does aₙ = (-1)ⁿ converge?

Step 1: Write out some terms

-1, 1, -1, 1, -1, ...

Step 2: Check the limit

The terms alternate between -1 and 1. They don't approach a single value.

Step 3: Conclusion

The sequence diverges (it oscillates).

Example 3: Does aₙ = (n+1)/n converge?

Step 1: Simplify

aₙ = (n+1)/n = 1 + 1/n

Step 2: Find the limit

lim(n→∞) (1 + 1/n) = 1 + 0 = 1

Step 3: Conclusion

The sequence converges to 1.

📋 How to Solve Most Questions

Step-by-Step: Determining Convergence

  1. Write out the formula — What is aₙ?
  2. Simplify if possible — Cancel terms, factor, etc.
  3. Find the limit — Use limit laws and techniques
  4. Check the limit — Does it exist and is it finite?
  5. State your conclusion — "Converges to L" or "Diverges"

Step-by-Step: Common Techniques

  1. Divide by highest power — For rational expressions
  2. Use known limits — lim(1/n) = 0, lim(rⁿ) = 0 if |r| < 1
  3. Check for oscillation — Does it alternate without settling?
  4. Check for unbounded growth — Does it go to infinity?

Convergence Tests

Chapter 2: Sequences and Series Lesson 3 of 7 ⏱️ 40-50 minutes

🎯 Goal

Learn various tests to determine whether a series converges or diverges.

In one sentence: Convergence tests are tools that help us determine if an infinite series adds up to a finite number.

💡 The Idea

Why Do We Need Tests?

Not all series converge. We need systematic ways to determine convergence without actually computing the sum.

The Divergence Test

If lim(n→∞) aₙ ≠ 0, then ∑aₙ diverges.

Example

For ∑(n=1 to ∞) n/(n+1):

lim(n→∞) n/(n+1) = 1 ≠ 0

The series diverges.

The Integral Test

If f is positive, continuous, and decreasing, then ∑f(n) and ∫f(x)dx either both converge or both diverge.

Example

For ∑(n=1 to ∞) 1/n²:

∫₁^∞ 1/x² dx = 1, which converges.

So the series converges.

Comparison Test

If 0 ≤ aₙ ≤ bₙ for all n:

  • If ∑bₙ converges, then ∑aₙ converges
  • If ∑aₙ diverges, then ∑bₙ diverges

Limit Comparison Test

If lim(n→∞) aₙ/bₙ = L where 0 < L < ∞, then ∑aₙ and ∑bₙ either both converge or both diverge.

Ratio Test

Let L = lim(n→∞) |aₙ₊₁/aₙ|

  • If L < 1, the series converges
  • If L > 1, the series diverges
  • If L = 1, the test is inconclusive

Root Test

Let L = lim(n→∞) ⁿ√|aₙ|

  • If L < 1, the series converges
  • If L > 1, the series diverges
  • If L = 1, the test is inconclusive

Why This Matters

Convergence tests are essential tools for analyzing infinite series. They allow us to determine convergence without computing the actual sum, which is often impossible.

📝 Example

Example 1: Using the Ratio Test

Problem: Does ∑(n=1 to ∞) n!/nⁿ converge?

Step 1: Set up the ratio

|aₙ₊₁/aₙ| = [(n+1)!/(n+1)^(n+1)] / [n!/nⁿ]

Step 2: Simplify

= [(n+1)!·nⁿ] / [n!·(n+1)^(n+1)]
= nⁿ / (n+1)ⁿ
= [n/(n+1)]ⁿ

Step 3: Take the limit

L = lim(n→∞) [n/(n+1)]ⁿ = 1/e < 1

Step 4: Conclusion

Since L < 1, the series converges.

Example 2: Using the Comparison Test

Problem: Does ∑(n=1 to ∞) 1/(n² + 1) converge?

Step 1: Find a comparison series

Compare with ∑1/n² (a convergent p-series).

Step 2: Set up the inequality

1/(n² + 1) < 1/n² for all n ≥ 1

Step 3: Apply the test

Since ∑1/n² converges and 0 ≤ 1/(n² + 1) < 1/n², our series converges.

Example 3: Using the Integral Test

Problem: Does ∑(n=1 to ∞) 1/n converge?

Step 1: Set up the integral

∫₁^∞ 1/x dx = lim(b→∞) [ln|x|]₁^b = lim(b→∞) ln(b) = ∞

Step 2: Conclusion

Since the integral diverges, the series diverges.

📋 How to Solve Most Questions

Step-by-Step: Choosing a Convergence Test

  1. Check the Divergence Test first — Does lim aₙ = 0?
  2. Look for factorials or exponentials — Try the Ratio Test
  3. Look for nth powers — Try the Root Test
  4. Compare to known series — Use Comparison or Limit Comparison
  5. Can you integrate? — Try the Integral Test

Step-by-Step: Applying the Ratio Test

  1. Write aₙ₊₁/aₙ — Set up the ratio
  2. Simplify — Cancel common terms
  3. Take the limit — Find L = lim|aₙ₊₁/aₙ|
  4. Compare to 1 — Conclude based on L

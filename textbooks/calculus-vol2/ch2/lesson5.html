Taylor Polynomials

Chapter 2: Sequences and Series Lesson 5 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to approximate functions using Taylor polynomials and understand how they work.

In one sentence: Taylor polynomials are polynomials that approximate a function by matching its derivatives at a point.

💡 The Idea

What is a Taylor Polynomial?

A Taylor polynomial of degree n for f(x) centered at x = a is:

Pₙ(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + ... + f⁽ⁿ⁾(a)(x-a)ⁿ/n!

This polynomial matches f and its first n derivatives at x = a.

Maclaurin Polynomials

A Maclaurin polynomial is a Taylor polynomial centered at a = 0:

Pₙ(x) = f(0) + f'(0)x + f''(0)x²/2! + ... + f⁽ⁿ⁾(0)xⁿ/n!

Common Taylor Polynomials

Exponential Function

eˣ ≈ 1 + x + x²/2! + x³/3! + ... + xⁿ/n!

Sine Function

sin(x) ≈ x - x³/3! + x⁵/5! - ... + (-1)ⁿx²ⁿ⁺¹/(2n+1)!

Cosine Function

cos(x) ≈ 1 - x²/2! + x⁴/4! - ... + (-1)ⁿx²ⁿ/(2n)!

Why Taylor Polynomials Work

Taylor polynomials work because they match the function's behavior at a point. The more derivatives we match, the better the approximation near that point.

Why This Matters

Taylor polynomials are used in calculators and computers to approximate functions. They're also the foundation for Taylor series, which represent functions exactly.

📝 Example

Example 1: Finding a Taylor Polynomial

Problem: Find the 3rd degree Taylor polynomial for f(x) = eˣ centered at a = 0

Step 1: Find derivatives

f(x) = eˣ → f(0) = 1
f'(x) = eˣ → f'(0) = 1
f''(x) = eˣ → f''(0) = 1
f'''(x) = eˣ → f'''(0) = 1

Step 2: Build the polynomial

P₃(x) = 1 + 1·x + 1·x²/2! + 1·x³/3!
= 1 + x + x²/2 + x³/6

Example 2: Approximating a Value

Problem: Approximate e⁰·¹ using the 3rd degree Taylor polynomial

Step 1: Use the polynomial from Example 1

P₃(x) = 1 + x + x²/2 + x³/6

Step 2: Substitute x = 0.1

P₃(0.1) = 1 + 0.1 + (0.1)²/2 + (0.1)³/6
= 1 + 0.1 + 0.005 + 0.000167
≈ 1.105167

Step 3: Compare to actual value

e⁰·¹ ≈ 1.105171

The approximation is very close!

Example 3: Finding a Taylor Polynomial for sin(x)

Problem: Find the 5th degree Taylor polynomial for sin(x) centered at a = 0

Step 1: Find derivatives

f(x) = sin(x) → f(0) = 0
f'(x) = cos(x) → f'(0) = 1
f''(x) = -sin(x) → f''(0) = 0
f'''(x) = -cos(x) → f'''(0) = -1
f⁴(x) = sin(x) → f⁴(0) = 0
f⁵(x) = cos(x) → f⁵(0) = 1

Step 2: Build the polynomial

P₅(x) = 0 + 1·x + 0·x²/2! + (-1)·x³/3! + 0·x⁴/4! + 1·x⁵/5!
= x - x³/6 + x⁵/120

📋 How to Solve Most Questions

Step-by-Step: Finding a Taylor Polynomial

  1. Find derivatives — Compute f, f', f'', ..., f⁽ⁿ⁾
  2. Evaluate at a — Find f(a), f'(a), f''(a), ..., f⁽ⁿ⁾(a)
  3. Build the polynomial — Use the Taylor polynomial formula
  4. Simplify — Combine like terms

Step-by-Step: Approximating Values

  1. Find the Taylor polynomial — Use the steps above
  2. Substitute x — Plug in the value you want to approximate
  3. Calculate — Evaluate the polynomial
  4. Compare — Check against the actual value if known

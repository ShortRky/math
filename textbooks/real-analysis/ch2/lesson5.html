Bolzano-Weierstrass Theorem

Chapter 2: Sequences Lesson 5 of 6 ⏱️ 40-50 minutes

🎯 Goal

Understand and apply the Bolzano-Weierstrass Theorem — one of the most important results in analysis.

In one sentence: Every bounded sequence has a convergent subsequence.

💡 The Idea

The Theorem

Bolzano-Weierstrass Theorem: Every bounded sequence of real numbers has a convergent subsequence.

This is a powerful existence theorem — it guarantees that convergent subsequences exist, even if we can't always find them explicitly.

Why It's True

The proof uses the completeness of ℝ and the bisection method:

  1. Start with a bounded sequence in an interval [a, b]
  2. Bisect the interval into [a, (a+b)/2] and [(a+b)/2, b]
  3. At least one half contains infinitely many terms
  4. Choose that half and repeat
  5. The nested intervals converge to a point L
  6. Construct a subsequence converging to L

Connection to Other Results

  • Monotone Convergence: Bounded monotone sequences converge
  • Cauchy Sequences: Bounded sequences have Cauchy subsequences
  • Compactness: In ℝ, bounded closed sets are compact

Applications

  • Proving the Extreme Value Theorem
  • Proving the Intermediate Value Theorem
  • Establishing properties of continuous functions
  • Proving convergence of iterative methods

Why This Matters

The Bolzano-Weierstrass Theorem is a cornerstone of analysis. It bridges the gap between boundedness and convergence, allowing us to extract convergent behavior from bounded sequences. This is essential for proving many fundamental theorems.

📝 Example

Example 1: Finding a Convergent Subsequence

Problem: The sequence aₙ = (-1)ⁿ is bounded. Find a convergent subsequence.

Step 1: Verify boundedness

The sequence is -1, 1, -1, 1, ... which is bounded by -1 and 1.

Step 2: Apply Bolzano-Weierstrass

By the theorem, there exists a convergent subsequence.

Step 3: Find the subsequence

Take the even-indexed terms: a₂ₙ = 1 for all n. This subsequence converges to 1.

Step 4: Alternative

Take the odd-indexed terms: a₂ₙ₋₁ = -1 for all n. This subsequence converges to -1.

Example 2: A More Complex Sequence

Problem: The sequence aₙ = sin(n) is bounded. What can we say about subsequences?

Step 1: Verify boundedness

Since -1 ≤ sin(n) ≤ 1 for all n, the sequence is bounded.

Step 2: Apply Bolzano-Weierstrass

There exists a convergent subsequence sin(nₖ) → L for some L ∈ [-1, 1].

Step 3: Interpretation

We can't easily find the subsequence, but we know it exists. The values sin(n) are dense in [-1, 1], so we can find subsequences converging to any value in [-1, 1].

📋 How to Solve Most Questions

Step-by-Step: Using Bolzano-Weierstrass

  1. Verify boundedness — Is the sequence bounded?
  2. Apply the theorem — A convergent subsequence exists
  3. Find the subsequence (if possible) — Look for patterns
  4. Use the result — Apply to prove other theorems

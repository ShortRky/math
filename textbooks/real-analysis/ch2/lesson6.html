Cauchy Sequences

Chapter 2: Sequences Lesson 6 of 6 ⏱️ 40-50 minutes

🎯 Goal

Understand Cauchy sequences and the Cauchy criterion — a powerful alternative to the ε-N definition of convergence.

In one sentence: A sequence is Cauchy if its terms get arbitrarily close to each other, and in ℝ, Cauchy sequences are exactly the convergent sequences.

💡 The Idea

What is a Cauchy Sequence?

A sequence (aₙ) is Cauchy if for every ε > 0, there exists N such that for all m, n > N:

|aₘ - aₙ| < ε

In other words, the terms eventually get arbitrarily close to each other.

Cauchy vs. Convergent

Key Theorem: In ℝ, a sequence is Cauchy if and only if it converges.

This is special to complete spaces. In ℚ, there are Cauchy sequences that don't converge (like sequences approaching √2).

Why Cauchy Sequences Matter

  • Don't need to know the limit: You can prove convergence without finding the limit
  • Works in abstract spaces: The Cauchy criterion generalizes to metric spaces
  • Practical: Often easier to check than the ε-N definition

Properties of Cauchy Sequences

  • Every Cauchy sequence is bounded
  • Every convergent sequence is Cauchy
  • Every Cauchy sequence converges (in ℝ)
  • Subsequences of Cauchy sequences are Cauchy

Why This Matters

Cauchy sequences provide a way to prove convergence without knowing the limit. This is essential in abstract settings where limits may not be easily computable. The Cauchy criterion is the foundation for completeness in metric spaces.

📝 Example

Example 1: Proving a Sequence is Cauchy

Problem: Prove that aₙ = 1/n is Cauchy

Step 1: Set up the Cauchy condition

Let ε > 0. We need to find N such that for all m, n > N:

|1/m - 1/n| < ε

Step 2: Simplify the expression

Assume m, n > N. Then:

|1/m - 1/n| ≤ |1/m| + |1/n| = 1/m + 1/n < 1/N + 1/N = 2/N

Step 3: Choose N

We need 2/N < ε, so choose N > 2/ε.

Step 4: Verify

For m, n > N, we have |1/m - 1/n| < 2/N < ε. Therefore, (1/n) is Cauchy.

Example 2: Using Cauchy to Prove Convergence

Problem: Prove that aₙ = (n+1)/n converges

Step 1: Show it's Cauchy

For m, n > N:

|(m+1)/m - (n+1)/n| = |1 + 1/m - 1 - 1/n| = |1/m - 1/n|

This is the same as Example 1, so it's Cauchy.

Step 2: Apply the theorem

Since (aₙ) is Cauchy and ℝ is complete, (aₙ) converges.

Step 3: Find the limit (optional)

We can see that (n+1)/n = 1 + 1/n → 1.

📋 How to Solve Most Questions

Step-by-Step: Working with Cauchy Sequences

  1. Understand the definition — Terms get close to each other
  2. Set up the inequality — |aₘ - aₙ| < ε
  3. Find N — Use the Archimedean property
  4. Verify — Check that the condition holds
  5. Apply the theorem — Cauchy ⟹ convergent (in ℝ)

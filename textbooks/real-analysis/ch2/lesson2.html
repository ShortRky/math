Limit Theorems

Chapter 2: Sequences Lesson 2 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn the fundamental theorems that allow us to compute limits of sums, products, and quotients of convergent sequences.

In one sentence: Limits respect arithmetic operations — the limit of a sum is the sum of the limits, and similarly for products and quotients.

💡 The Idea

Why Limit Theorems Matter

Computing limits from the ε definition is tedious. Limit theorems give us shortcuts — we can break complex limits into simpler ones.

The Main Theorems

If lim aₙ = A and lim bₙ = B, then:

lim(aₙ + bₙ) = A + B
lim(aₙ - bₙ) = A - B
lim(aₙ · bₙ) = A · B
lim(aₙ / bₙ) = A / B (if B ≠ 0)

Additional Useful Results

  • Squeeze Theorem: If aₙ ≤ cₙ ≤ bₙ and lim aₙ = lim bₙ = L, then lim cₙ = L
  • Comparison: If 0 ≤ aₙ ≤ bₙ and lim bₙ = 0, then lim aₙ = 0
  • Powers: lim(aₙᵏ) = (lim aₙ)ᵏ for any positive integer k
  • Roots: lim(ᵏ√aₙ) = ᵏ√(lim aₙ) if the limit is positive

Proof Strategy

To prove these theorems, we use the triangle inequality and the ε definition. For example, to prove lim(aₙ + bₙ) = A + B:

|(aₙ + bₙ) - (A + B)| ≤ |aₙ - A| + |bₙ - B|

We can make each term small by choosing n large enough.

Why This Matters

Limit theorems are the workhorses of analysis. They allow us to compute limits without returning to the ε definition every time. Every calculus rule you've used depends on these theorems.

📝 Example

Example 1: Computing a Limit Using Theorems

Problem: Find lim(n→∞) (3n² + 2n)/(n² + 1)

Step 1: Divide numerator and denominator by n²

(3n² + 2n)/(n² + 1) = (3 + 2/n)/(1 + 1/n²)

Step 2: Apply limit theorems

As n → ∞, 2/n → 0 and 1/n² → 0

Step 3: Compute the limit

lim(3 + 2/n)/(1 + 1/n²) = (3 + 0)/(1 + 0) = 3

Example 2: Using the Squeeze Theorem

Problem: Prove that lim(n→∞) sin(n)/n = 0

Step 1: Find bounds

For all n, -1 ≤ sin(n) ≤ 1

Step 2: Divide by n

-1/n ≤ sin(n)/n ≤ 1/n

Step 3: Apply Squeeze Theorem

Since lim(-1/n) = 0 and lim(1/n) = 0, we have lim(sin(n)/n) = 0

📋 How to Solve Most Questions

Step-by-Step: Using Limit Theorems

  1. Identify the form — Is it a sum, product, quotient?
  2. Check conditions — Do the individual limits exist?
  3. Apply the appropriate theorem — Sum, product, or quotient rule
  4. Simplify — Reduce to known limits
  5. Verify — Check that the result makes sense

Convergence of Sequences

Chapter 2: Sequences Lesson 1 of 6 ⏱️ 35-45 minutes

🎯 Goal

Understand the rigorous definition of sequence convergence and learn how to prove that sequences converge.

In one sentence: A sequence converges to a limit L if, for any ε > 0, the terms eventually stay within ε of L.

💡 The Idea

What is a Sequence?

A sequence is an ordered list of numbers: a₁, a₂, a₃, ...

We write this as (aₙ) or {aₙ}ₙ₌₁^∞. Each aₙ is a term of the sequence.

The ε Definition of Convergence

A sequence (aₙ) converges to a limit L if:

For every ε > 0, there exists N ∈ ℕ such that for all n > N, |aₙ - L| < ε.

We write: lim(n→∞) aₙ = L or aₙ → L

Intuitive Meaning

Convergence means the terms get arbitrarily close to L and stay there. No matter how small you make ε (the "error tolerance"), eventually all terms are within ε of L.

The N tells you when this happens — after the Nth term, everything is close enough.

Examples of Convergent Sequences

  • 1/n → 0 (terms get closer to 0)
  • (n+1)/n → 1 (terms approach 1)
  • 1 + 1/2 + 1/4 + ... → 2 (geometric series)

Why This Matters

The ε definition is the foundation of all analysis. It makes the intuitive idea of "getting close" precise. Every theorem about limits, continuity, and derivatives builds on this definition.

📝 Example

Example 1: Prove that 1/n → 0

Problem: Prove that lim(n→∞) 1/n = 0

Step 1: Set up the ε definition

Let ε > 0. We need to find N such that for all n > N, |1/n - 0| < ε.

Step 2: Simplify the inequality

|1/n - 0| = 1/n < ε ⟹ n > 1/ε

Step 3: Choose N

By the Archimedean property, there exists N ∈ ℕ such that N > 1/ε.

Step 4: Verify

For n > N, we have n > 1/ε, so 1/n < ε. Therefore, |1/n - 0| < ε.

Step 5: Conclusion

Since ε was arbitrary, lim(n→∞) 1/n = 0.

Example 2: Prove that (2n+1)/n → 2

Problem: Prove that lim(n→∞) (2n+1)/n = 2

Step 1: Set up the ε definition

Let ε > 0. We need to find N such that for all n > N, |(2n+1)/n - 2| < ε.

Step 2: Simplify the inequality

|(2n+1)/n - 2| = |(2n+1-2n)/n| = |1/n| = 1/n < ε ⟹ n > 1/ε

Step 3: Choose N

Choose N ∈ ℕ such that N > 1/ε.

Step 4: Verify

For n > N, we have 1/n < 1/N < ε. Therefore, |(2n+1)/n - 2| < ε.

Step 5: Conclusion

Since ε was arbitrary, lim(n→∞) (2n+1)/n = 2.

📋 How to Solve Most Questions

Step-by-Step: Proving Convergence

  1. State the limit — What do you think the sequence converges to?
  2. Let ε > 0 — Start with an arbitrary positive error
  3. Simplify |aₙ - L| — Algebraically manipulate the difference
  4. Find N — Determine when the difference is less than ε
  5. Verify — Show that for n > N, the condition holds
  6. Conclude — Since ε was arbitrary, the limit exists

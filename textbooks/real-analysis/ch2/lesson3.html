Monotone Sequences

Chapter 2: Sequences Lesson 3 of 6 ⏱️ 35-45 minutes

🎯 Goal

Understand monotone sequences and the Monotone Convergence Theorem — a powerful tool for proving convergence without knowing the limit in advance.

In one sentence: Every bounded monotone sequence converges — if a sequence always increases (or always decreases) and is bounded, it must converge.

💡 The Idea

What is a Monotone Sequence?

A sequence (aₙ) is:

  • Increasing: aₙ ≤ aₙ₊₁ for all n
  • Decreasing: aₙ ≥ aₙ₊₁ for all n
  • Monotone: if it's either increasing or decreasing

The Monotone Convergence Theorem

Theorem: Every bounded monotone sequence converges.

More specifically:

  • If (aₙ) is increasing and bounded above, then it converges to sup{aₙ}
  • If (aₙ) is decreasing and bounded below, then it converges to inf{aₙ}

Why This Matters

This theorem is powerful because:

  • You don't need to know the limit in advance
  • You only need to check two things: monotonicity and boundedness
  • It's often easier than using the ε definition directly

Proof Sketch

For an increasing sequence bounded above:

  1. By completeness, the set {aₙ} has a supremum L
  2. For any ε > 0, L - ε is not an upper bound
  3. So there exists N with aₙ > L - ε
  4. Since the sequence is increasing, for all n > N, L - ε < aₙ ≤ L
  5. Therefore |aₙ - L| < ε for all n > N

Why This Matters

The Monotone Convergence Theorem is one of the most practical tools in analysis. It lets you prove convergence by checking just two simple properties, without ever computing the limit explicitly.

📝 Example

Example 1: Proving Convergence of a Monotone Sequence

Problem: Prove that aₙ = 1 - 1/n converges

Step 1: Check monotonicity

aₙ₊₁ - aₙ = (1 - 1/(n+1)) - (1 - 1/n) = 1/n - 1/(n+1) = 1/(n(n+1)) > 0

So the sequence is increasing.

Step 2: Check boundedness

For all n: aₙ = 1 - 1/n < 1

So the sequence is bounded above by 1.

Step 3: Apply the theorem

Since (aₙ) is increasing and bounded above, it converges.

Step 4: Find the limit (optional)

The limit is sup{aₙ} = 1

Example 2: A Decreasing Sequence

Problem: Prove that aₙ = 1/2ⁿ converges

Step 1: Check monotonicity

aₙ₊₁ = 1/2ⁿ⁺¹ = (1/2)(1/2ⁿ) = aₙ/2 < aₙ

So the sequence is decreasing.

Step 2: Check boundedness

For all n: aₙ = 1/2ⁿ > 0

So the sequence is bounded below by 0.

Step 3: Apply the theorem

Since (aₙ) is decreasing and bounded below, it converges.

Step 4: Find the limit (optional)

The limit is inf{aₙ} = 0

📋 How to Solve Most Questions

Step-by-Step: Using the Monotone Convergence Theorem

  1. Check monotonicity — Is aₙ₊₁ ≥ aₙ or aₙ₊₁ ≤ aₙ?
  2. Check boundedness — Is there an upper/lower bound?
  3. Apply the theorem — If both hold, the sequence converges
  4. Find the limit (if needed) — It's the sup or inf of the set

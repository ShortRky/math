Subsequences

Chapter 2: Sequences Lesson 4 of 6 ⏱️ 35-45 minutes

🎯 Goal

Understand subsequences and their relationship to convergence — a crucial concept for the Bolzano-Weierstrass Theorem.

In one sentence: A subsequence is a sequence formed by selecting terms from another sequence, and if a sequence converges, all its subsequences converge to the same limit.

💡 The Idea

What is a Subsequence?

Given a sequence (aₙ), a subsequence is a sequence (aₙₖ) where n₁ < n₂ < n₃ < ... are indices from the original sequence.

For example, if aₙ = 1/n, then a₂ₖ = 1/(2k) gives the subsequence 1/2, 1/4, 1/6, ...

Key Property

Theorem: If aₙ → L, then every subsequence aₙₖ also converges to L.

This makes sense: if the original sequence gets close to L and stays there, then any selection of terms from that sequence also gets close to L and stays there.

The Converse is False

If every subsequence converges to L, does the original sequence converge to L? Yes!

But if some subsequence converges to L, the original sequence might not converge. For example, aₙ = (-1)ⁿ has subsequences converging to 1 and -1, but the sequence itself doesn't converge.

Why Subsequences Matter

  • They help us understand the behavior of sequences
  • They're essential for the Bolzano-Weierstrass Theorem
  • They allow us to extract convergent subsequences from bounded sequences

Why This Matters

Subsequences are a powerful tool for analyzing sequences. They allow us to "zoom in" on specific parts of a sequence and extract convergent behavior, even when the original sequence doesn't converge.

📝 Example

Example 1: Subsequences of a Convergent Sequence

Problem: Let aₙ = 1/n. Show that every subsequence converges to 0.

Step 1: Note that aₙ → 0

We know that 1/n → 0 as n → ∞.

Step 2: Consider any subsequence

Let aₙₖ = 1/nₖ be any subsequence. Since nₖ ≥ k, we have:

0 < 1/nₖ ≤ 1/k

Step 3: Apply the Squeeze Theorem

Since 1/k → 0, by the Squeeze Theorem, 1/nₖ → 0.

Step 4: Conclusion

Every subsequence of (1/n) converges to 0.

Example 2: A Sequence with Different Subsequential Limits

Problem: Let aₙ = (-1)ⁿ. Find two subsequences with different limits.

Step 1: Identify the sequence

The sequence is: -1, 1, -1, 1, -1, 1, ...

Step 2: Find a subsequence converging to 1

Take nₖ = 2k (even indices): a₂ₖ = (-1)²ᵏ = 1 → 1

Step 3: Find a subsequence converging to -1

Take nₖ = 2k-1 (odd indices): a₂ₖ₋₁ = (-1)²ᵏ⁻¹ = -1 → -1

Step 4: Conclusion

The sequence has subsequences converging to different limits, so the original sequence doesn't converge.

📋 How to Solve Most Questions

Step-by-Step: Working with Subsequences

  1. Identify the original sequence — What is (aₙ)?
  2. Define the subsequence — Choose indices n₁ < n₂ < n₃ < ...
  3. Write out the subsequence — What is aₙₖ?
  4. Analyze convergence — Does it converge? To what?
  5. Draw conclusions — What does this tell you about the original sequence?

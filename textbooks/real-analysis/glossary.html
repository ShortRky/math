Glossary of Terms

Real Analysis Reference Guide

📖 Chapter 1: The Real Numbers

Axiom

A statement assumed to be true without proof; forms the foundation of a mathematical system.

Field Axioms

The algebraic properties that define addition and multiplication: closure, commutativity, associativity, identity, inverse, and distributivity.

Order Axioms

Properties that define the ordering of real numbers: trichotomy, transitivity, and compatibility with addition and multiplication.

Completeness Axiom

Every non-empty set of real numbers that is bounded above has a least upper bound (supremum) in ℝ.

Upper Bound

A number M such that x ≤ M for all x in a set S.

Lower Bound

A number m such that m ≤ x for all x in a set S.

Supremum (sup)

The least upper bound of a set — the smallest number that is greater than or equal to all elements in the set.

Infimum (inf)

The greatest lower bound of a set — the largest number that is less than or equal to all elements in the set.

Dense

A set A is dense in ℝ if between any two real numbers, there exists an element of A. For example, ℚ is dense in ℝ.

Archimedean Property

For any real number x, there exists a natural number n such that n > x. Equivalently, ℕ is not bounded above in ℝ.

📖 Chapter 2: Sequences

Sequence

An ordered list of numbers a₁, a₂, a₃, ... indexed by natural numbers. Written as (aₙ) or {aₙ}ₙ₌₁^∞.

Convergence

A sequence (aₙ) converges to a limit L if for every ε > 0, there exists N such that for all n > N, |aₙ - L| < ε.

Limit

The value L that a sequence approaches as n → ∞. Written as lim(n→∞) aₙ = L or aₙ → L.

Divergence

A sequence that does not converge to any finite limit.

Bounded Sequence

A sequence (aₙ) is bounded if there exists M > 0 such that |aₙ| ≤ M for all n.

Monotone Sequence

A sequence that is either increasing (aₙ ≤ aₙ₊₁) or decreasing (aₙ ≥ aₙ₊₁) for all n.

Monotone Convergence Theorem

Every bounded monotone sequence converges. Increasing bounded sequences converge to their supremum; decreasing bounded sequences converge to their infimum.

Subsequence

A sequence (aₙₖ) formed by selecting terms from another sequence (aₙ) where n₁ < n₂ < n₃ < ... are indices.

Bolzano-Weierstrass Theorem

Every bounded sequence has a convergent subsequence.

Cauchy Sequence

A sequence (aₙ) where for every ε > 0, there exists N such that for all m, n > N, |aₘ - aₙ| < ε. In ℝ, Cauchy sequences are exactly the convergent sequences.

Squeeze Theorem

If aₙ ≤ cₙ ≤ bₙ and lim aₙ = lim bₙ = L, then lim cₙ = L.

📖 Chapter 3: Continuity

Continuous Function

A function f is continuous at a point c if for every ε > 0, there exists δ > 0 such that for all x with |x - c| < δ, we have |f(x) - f(c)| < ε.

ε-δ Definition

The formal definition of continuity: small changes in input produce small changes in output.

Discontinuous

A function that is not continuous at a point. Types include removable, jump, and infinite discontinuities.

Intermediate Value Theorem (IVT)

If f is continuous on [a, b] and k is between f(a) and f(b), then there exists c ∈ [a, b] such that f(c) = k.

Extreme Value Theorem (EVT)

If f is continuous on a closed interval [a, b], then f attains its maximum and minimum values on [a, b].

Uniform Continuity

A function f is uniformly continuous on a set S if for every ε > 0, there exists δ > 0 (independent of the point) such that for all x, y ∈ S with |x - y| < δ, we have |f(x) - f(y)| < ε.

Heine-Cantor Theorem

Every continuous function on a closed interval [a, b] is uniformly continuous.

Lipschitz Continuity

A function f is Lipschitz continuous if there exists a constant K such that |f(x) - f(y)| ≤ K|x - y| for all x, y in the domain. Lipschitz continuity implies uniform continuity.

Composition of Continuous Functions

If g is continuous at c and f is continuous at g(c), then f ∘ g is continuous at c.

📐 Common Symbols

ℝ

The set of real numbers.

ℚ

The set of rational numbers (fractions p/q where p, q are integers and q ≠ 0).

ℕ

The set of natural numbers {1, 2, 3, ...}.

ℤ

The set of integers {..., -2, -1, 0, 1, 2, ...}.

∈

"Is an element of" — x ∈ S means x is in the set S.

∀

"For all" or "for every" — universal quantifier.

∃

"There exists" — existential quantifier.

ε (epsilon)

A small positive number, often used to represent an error tolerance.

δ (delta)

A small positive number, often used to represent an input tolerance.

→

"Converges to" — aₙ → L means the sequence aₙ converges to L.

∞

Infinity — used in limits to indicate unbounded behavior.

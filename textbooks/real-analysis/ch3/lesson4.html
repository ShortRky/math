Extreme Value Theorem

Chapter 3: Continuity Lesson 4 of 5 ⏱️ 35-45 minutes

🎯 Goal

Understand and apply the Extreme Value Theorem — continuous functions on closed intervals attain maximum and minimum values.

In one sentence: A continuous function on a closed interval must achieve both a maximum and a minimum value.

💡 The Idea

The Theorem

Extreme Value Theorem (EVT): If f is continuous on a closed interval [a, b], then f attains its maximum and minimum values on [a, b].

In other words, there exist c, d ∈ [a, b] such that f(c) ≤ f(x) ≤ f(d) for all x ∈ [a, b].

Why Closed Intervals?

The theorem requires a closed interval [a, b], not an open interval (a, b).

Example: f(x) = 1/x on (0, 1) is continuous but has no minimum — it approaches 0 but never reaches it.

Proof Strategy

The proof uses two key results:

  1. Boundedness Theorem: Continuous functions on closed intervals are bounded
  2. Bolzano-Weierstrass: Every bounded sequence has a convergent subsequence

We show that f is bounded, then use sequences to find where the supremum and infimum are attained.

Applications

  • Optimization: Guarantees that maximum and minimum values exist
  • Physics: Energy, temperature, and other quantities have maximum/minimum values
  • Economics: Profit, cost, and utility functions have optimal values

Why This Matters

The EVT is essential for optimization problems. It guarantees that when you're looking for maximum or minimum values of a continuous function on a closed interval, they actually exist. This is crucial for calculus, physics, engineering, and economics.

📝 Example

Example 1: Finding Maximum and Minimum

Problem: Find the maximum and minimum of f(x) = x² - 4x + 3 on [0, 5]

Step 1: Verify conditions

f(x) = x² - 4x + 3 is a polynomial, so it's continuous on [0, 5].

Step 2: Find critical points

f'(x) = 2x - 4 = 0
x = 2

Step 3: Evaluate at critical points and endpoints

f(0) = 0² - 4(0) + 3 = 3
f(2) = 2² - 4(2) + 3 = 4 - 8 + 3 = -1
f(5) = 5² - 4(5) + 3 = 25 - 20 + 3 = 8

Step 4: Identify maximum and minimum

Maximum: f(5) = 8 at x = 5

Minimum: f(2) = -1 at x = 2

Example 2: Why Closed Intervals Matter

Problem: Show that f(x) = x on (0, 1) has no maximum

Step 1: Check continuity

f(x) = x is continuous on (0, 1).

Step 2: Look for maximum

For any x ∈ (0, 1), we have x < 1. But 1 is not in the interval (0, 1).

Step 3: Conclusion

f has no maximum on (0, 1) because the supremum (1) is not in the interval.

Note: If we consider [0, 1], then f(1) = 1 is the maximum.

📋 How to Solve Most Questions

Step-by-Step: Using the Extreme Value Theorem

  1. Check continuity — Is f continuous on [a, b]?
  2. Find critical points — Where is f'(x) = 0 or undefined?
  3. Evaluate at critical points — What is f(c) for each critical point?
  4. Evaluate at endpoints — What are f(a) and f(b)?
  5. Compare values — The largest is the maximum, smallest is the minimum

Uniform Continuity

Chapter 3: Continuity Lesson 5 of 5 ⏱️ 40-50 minutes

🎯 Goal

Understand uniform continuity — a stronger form of continuity that's essential for integration and advanced analysis.

In one sentence: A function is uniformly continuous if the same δ works for every point in the domain, not just at individual points.

💡 The Idea

The Definition

A function f is uniformly continuous on a set S if:

For every ε > 0, there exists δ > 0 such that for all x, y ∈ S with |x - y| < δ:

|f(x) - f(y)| < ε

The key difference from ordinary continuity: δ depends only on ε, not on the point x.

Ordinary vs. Uniform Continuity

Ordinary continuity at c: For every ε > 0, there exists δ > 0 (which may depend on c) such that if |x - c| < δ, then |f(x) - f(c)| < ε.

Uniform continuity on S: For every ε > 0, there exists δ > 0 (which works for all points in S) such that if |x - y| < δ, then |f(x) - f(y)| < ε.

Important Theorems

  • Heine-Cantor Theorem: Every continuous function on a closed interval [a, b] is uniformly continuous
  • Lipschitz continuity: If |f(x) - f(y)| ≤ K|x - y| for some constant K, then f is uniformly continuous
  • Composition: The composition of uniformly continuous functions is uniformly continuous

Why Uniform Continuity Matters

  • Integration: Uniformly continuous functions are Riemann integrable
  • Approximation: We can approximate uniformly continuous functions uniformly
  • Extension: Uniformly continuous functions can be extended to closures

Why This Matters

Uniform continuity is crucial for many advanced topics in analysis. It ensures that functions behave "consistently" across their entire domain, which is essential for integration, approximation theory, and extending functions to larger domains.

📝 Example

Example 1: Proving Uniform Continuity

Problem: Prove that f(x) = 2x + 1 is uniformly continuous on ℝ

Step 1: Set up the definition

Let ε > 0. We need to find δ > 0 such that for all x, y with |x - y| < δ:

|(2x + 1) - (2y + 1)| < ε

Step 2: Simplify

|(2x + 1) - (2y + 1)| = |2x - 2y| = 2|x - y|

Step 3: Choose δ

We need 2|x - y| < ε, so |x - y| < ε/2. Choose δ = ε/2.

Step 4: Verify

If |x - y| < δ = ε/2, then |f(x) - f(y)| = 2|x - y| < 2(ε/2) = ε.

Step 5: Conclusion

f(x) = 2x + 1 is uniformly continuous on ℝ.

Example 2: A Function That's Not Uniformly Continuous

Problem: Show that f(x) = 1/x is not uniformly continuous on (0, 1)

Step 1: Choose ε

Let ε = 1. We need to show that for any δ > 0, there exist x, y with |x - y| < δ but |f(x) - f(y)| ≥ 1.

Step 2: Find x, y for any δ

Given δ > 0, choose n large enough so that 1/n < δ. Let:

x = 1/n, y = 1/(n+1)

Step 3: Check |x - y|

|x - y| = |1/n - 1/(n+1)| = 1/(n(n+1)) < 1/n < δ

Step 4: Check |f(x) - f(y)|

|f(x) - f(y)| = |n - (n+1)| = 1 ≥ ε

Step 5: Conclusion

For any δ > 0, we found x, y with |x - y| < δ but |f(x) - f(y)| ≥ 1. Therefore, f is not uniformly continuous on (0, 1).

📋 How to Solve Most Questions

Step-by-Step: Proving Uniform Continuity

  1. Set up the definition — Let ε > 0, find δ > 0
  2. Simplify |f(x) - f(y)| — Express in terms of |x - y|
  3. Choose δ — Often δ = ε/K for some constant K
  4. Verify — Show that |x - y| < δ implies |f(x) - f(y)| < ε
  5. Conclude — The function is uniformly continuous

Epsilon-Delta Definition

Chapter 3: Continuity Lesson 1 of 5 ⏱️ 40-50 minutes

🎯 Goal

Master the rigorous ε-δ definition of continuity — the foundation for all of analysis.

In one sentence: A function is continuous at a point if, for any error tolerance ε, we can find an input tolerance δ that keeps the output within ε.

💡 The Idea

The Definition

A function f is continuous at a point c if:

For every ε > 0, there exists δ > 0 such that for all x with |x - c| < δ:

|f(x) - f(c)| < ε

f is continuous on a set if it's continuous at every point in that set.

Intuitive Meaning

Continuity means "small changes in input produce small changes in output."

The ε-δ definition makes this precise: if you want the output to be within ε of f(c), you can find a δ such that whenever the input is within δ of c, the output is within ε of f(c).

Connection to Limits

f is continuous at c if and only if:

  1. f(c) is defined
  2. lim(x→c) f(x) exists
  3. lim(x→c) f(x) = f(c)

The ε-δ definition is equivalent to the sequential definition: f is continuous at c if and only if for every sequence xₙ → c, we have f(xₙ) → f(c).

Why This Definition?

  • Precise: No ambiguity about what "close" means
  • Quantifiable: We can measure how continuous a function is
  • Generalizable: Works in any metric space
  • Foundation: All theorems about continuous functions depend on this

Why This Matters

The ε-δ definition is the heart of analysis. It transforms the intuitive idea of continuity into a precise mathematical statement. Every theorem about continuous functions — the Intermediate Value Theorem, Extreme Value Theorem, and more — depends on this definition.

📝 Example

Example 1: Proving Continuity of f(x) = x² at c = 2

Problem: Prove that f(x) = x² is continuous at c = 2

Step 1: Set up the definition

Let ε > 0. We need to find δ > 0 such that if |x - 2| < δ, then |x² - 4| < ε.

Step 2: Simplify |x² - 4|

|x² - 4| = |x - 2||x + 2|

Step 3: Bound |x + 2|

If |x - 2| < 1, then 1 < x < 3, so 3 < x + 2 < 5. Thus |x + 2| < 5.

Step 4: Choose δ

We need |x - 2||x + 2| < ε. Since |x + 2| < 5 when |x - 2| < 1, we need 5|x - 2| < ε, so |x - 2| < ε/5.

Choose δ = min(1, ε/5).

Step 5: Verify

If |x - 2| < δ, then |x - 2| < 1 (so |x + 2| < 5) and |x - 2| < ε/5. Therefore:

|x² - 4| = |x - 2||x + 2| < (ε/5) · 5 = ε

Example 2: A Discontinuous Function

Problem: Show that f(x) = 1/x is not continuous at c = 0

Step 1: Check the definition

f(0) is not defined, so f cannot be continuous at 0.

Step 2: Even if we define f(0)

Suppose we define f(0) = a for some a. For any δ > 0, we can find x with |x| < δ but |1/x| is arbitrarily large. So we can't make |f(x) - a| < ε for small ε.

Step 3: Conclusion

f(x) = 1/x is not continuous at 0.

📋 How to Solve Most Questions

Step-by-Step: Proving Continuity with ε-δ

  1. State the goal — For every ε > 0, find δ > 0
  2. Let ε > 0 — Start with an arbitrary positive error
  3. Work backwards — Simplify |f(x) - f(c)| to find what δ should be
  4. Choose δ — Often depends on ε and bounds on f
  5. Verify — Show that |x - c| < δ implies |f(x) - f(c)| < ε

Properties of Continuous Functions

Chapter 3: Continuity Lesson 2 of 5 ⏱️ 35-45 minutes

🎯 Goal

Learn the fundamental properties of continuous functions — how continuity interacts with arithmetic operations and composition.

In one sentence: Continuous functions can be added, multiplied, and composed to create new continuous functions.

💡 The Idea

Arithmetic Properties

If f and g are continuous at c, then:

  • Sum: f + g is continuous at c
  • Difference: f - g is continuous at c
  • Product: f · g is continuous at c
  • Quotient: f / g is continuous at c (if g(c) ≠ 0)
  • Scalar multiple: kf is continuous at c for any constant k

Composition

If g is continuous at c and f is continuous at g(c), then f ∘ g is continuous at c.

In other words: the composition of continuous functions is continuous.

Why These Properties Hold

These follow from the limit theorems. For example, to show f + g is continuous:

lim(x→c) [f(x) + g(x)] = lim(x→c) f(x) + lim(x→c) g(x) = f(c) + g(c)

The other properties follow similarly.

Examples of Continuous Functions

  • Polynomials: p(x) = aₙxⁿ + ... + a₁x + a₀ (continuous everywhere)
  • Rational functions: p(x)/q(x) (continuous where q(x) ≠ 0)
  • Trigonometric functions: sin(x), cos(x) (continuous everywhere)
  • Exponential functions: eˣ (continuous everywhere)
  • Logarithmic functions: ln(x) (continuous for x > 0)

Why This Matters

These properties allow us to build complex continuous functions from simple ones. Instead of proving continuity from scratch for every function, we can use these rules to quickly determine continuity.

📝 Example

Example 1: Proving Continuity Using Properties

Problem: Show that f(x) = (x² + 1)/(x + 2) is continuous at x = 1

Step 1: Identify the components

f(x) = (x² + 1)/(x + 2) is a quotient of two functions:

  • g(x) = x² + 1 (polynomial, continuous everywhere)
  • h(x) = x + 2 (polynomial, continuous everywhere)

Step 2: Check the denominator

h(1) = 1 + 2 = 3 ≠ 0

Step 3: Apply the quotient rule

Since g and h are continuous at x = 1 and h(1) ≠ 0, the quotient g/h is continuous at x = 1.

Step 4: Conclusion

f(x) = (x² + 1)/(x + 2) is continuous at x = 1.

Example 2: Composition of Continuous Functions

Problem: Show that f(x) = sin(x²) is continuous everywhere

Step 1: Identify the composition

f(x) = sin(x²) = (sin ∘ g)(x) where g(x) = x²

Step 2: Check continuity of components

  • g(x) = x² is continuous everywhere (polynomial)
  • sin(x) is continuous everywhere

Step 3: Apply the composition rule

Since g is continuous everywhere and sin is continuous everywhere, the composition sin ∘ g is continuous everywhere.

Step 4: Conclusion

f(x) = sin(x²) is continuous everywhere.

📋 How to Solve Most Questions

Step-by-Step: Using Properties to Prove Continuity

  1. Break down the function — Identify sums, products, quotients, compositions
  2. Check each component — Are the basic functions continuous?
  3. Apply the appropriate rule — Sum, product, quotient, or composition
  4. Check conditions — For quotients, is the denominator nonzero?
  5. Conclude — The function is continuous where the rules apply

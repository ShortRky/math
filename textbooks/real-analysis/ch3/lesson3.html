Intermediate Value Theorem

Chapter 3: Continuity Lesson 3 of 5 ⏱️ 35-45 minutes

🎯 Goal

Understand and apply the Intermediate Value Theorem — a fundamental result about the behavior of continuous functions.

In one sentence: A continuous function on an interval takes on every value between its endpoint values.

💡 The Idea

The Theorem

Intermediate Value Theorem (IVT): If f is continuous on [a, b] and k is any value between f(a) and f(b), then there exists c ∈ [a, b] such that f(c) = k.

In other words: continuous functions can't "skip" values.

Intuitive Meaning

If you draw the graph of a continuous function without lifting your pen, you must hit every height between the starting and ending points.

For example, if f(1) = 3 and f(5) = 7, then f must take on every value between 3 and 7 somewhere in [1, 5].

Special Case: The Zero Theorem

If f is continuous on [a, b] and f(a) and f(b) have opposite signs, then there exists c ∈ (a, b) such that f(c) = 0.

This is useful for proving that equations have solutions.

Proof Sketch

The proof uses the completeness of ℝ:

  1. Assume f(a) < k < f(b)
  2. Let S = {x ∈ [a, b] : f(x) < k}
  3. S is non-empty (a ∈ S) and bounded above (by b)
  4. By completeness, c = sup(S) exists
  5. Use continuity to show f(c) = k

Why This Matters

The IVT is essential for proving that equations have solutions, that continuous functions have roots, and that certain values must be attained. It's used throughout mathematics and its applications.

📝 Example

Example 1: Proving a Root Exists

Problem: Show that x³ + x - 1 = 0 has a solution in (0, 1)

Step 1: Define the function

Let f(x) = x³ + x - 1. This is a polynomial, so it's continuous everywhere.

Step 2: Evaluate at the endpoints

f(0) = 0³ + 0 - 1 = -1 < 0
f(1) = 1³ + 1 - 1 = 1 > 0

Step 3: Apply the Zero Theorem

Since f is continuous on [0, 1] and f(0) < 0 < f(1), there exists c ∈ (0, 1) such that f(c) = 0.

Step 4: Conclusion

The equation x³ + x - 1 = 0 has a solution in (0, 1).

Example 2: Using IVT to Find Values

Problem: Show that cos(x) = x has a solution

Step 1: Define the function

Let f(x) = cos(x) - x. This is continuous everywhere.

Step 2: Evaluate at convenient points

f(0) = cos(0) - 0 = 1 > 0
f(π/2) = cos(π/2) - π/2 = 0 - π/2 = -π/2 < 0

Step 3: Apply the Zero Theorem

Since f is continuous on [0, π/2] and f(0) > 0 > f(π/2), there exists c ∈ (0, π/2) such that f(c) = 0.

Step 4: Conclusion

There exists c ∈ (0, π/2) such that cos(c) = c.

📋 How to Solve Most Questions

Step-by-Step: Using the Intermediate Value Theorem

  1. Define the function — What function are you analyzing?
  2. Check continuity — Is the function continuous on the interval?
  3. Evaluate at endpoints — What are f(a) and f(b)?
  4. Check the condition — Is k between f(a) and f(b)?
  5. Apply the theorem — There exists c with f(c) = k

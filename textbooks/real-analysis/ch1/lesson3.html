Supremum and Infimum

Chapter 1: The Real Numbers Lesson 3 of 5 ⏱️ 35-45 minutes

🎯 Goal

Master the concepts of supremum (least upper bound) and infimum (greatest lower bound) and learn how to compute them.

In one sentence: The supremum is the smallest number that's greater than or equal to all elements in a set, while the infimum is the largest number that's less than or equal to all elements.

💡 The Idea

Upper Bounds and Supremum

An upper bound M for a set S satisfies: for all x ∈ S, x ≤ M.

The supremum (sup) is the least upper bound — the smallest number that's still an upper bound.

sup(S) = smallest M such that x ≤ M for all x ∈ S

Lower Bounds and Infimum

A lower bound m for a set S satisfies: for all x ∈ S, m ≤ x.

The infimum (inf) is the greatest lower bound — the largest number that's still a lower bound.

inf(S) = largest m such that m ≤ x for all x ∈ S

Key Properties

  • If S has a maximum, then sup(S) = max(S)
  • If S has a minimum, then inf(S) = min(S)
  • sup(S) may not be in S (e.g., sup{1, 1/2, 1/3, ...} = 1, but 1 is in S)
  • inf(S) may not be in S (e.g., inf{1, 1/2, 1/3, ...} = 0, but 0 is not in S)

Why This Matters

Supremum and infimum are essential for rigorous analysis. They allow us to talk about bounds precisely, even when sets don't have maximums or minimums. This is crucial for convergence, continuity, and integration.

📝 Example

Example 1: Finding Supremum and Infimum

Problem: Find sup and inf of S = {x ∈ ℝ : 0 < x < 5}

Step 1: Find upper bounds

Any number ≥ 5 is an upper bound. For example, 5, 6, 100 are all upper bounds.

Step 2: Find the least upper bound

5 is an upper bound. Is it the least? Yes, because any number less than 5 (like 4.9) is not an upper bound (4.95 ∈ S but 4.95 > 4.9).

Therefore, sup(S) = 5.

Step 3: Find lower bounds

Any number ≤ 0 is a lower bound. For example, 0, -1, -100 are all lower bounds.

Step 4: Find the greatest lower bound

0 is a lower bound. Is it the greatest? Yes, because any number greater than 0 (like 0.1) is not a lower bound (0.05 ∈ S but 0.05 < 0.1).

Therefore, inf(S) = 0.

Example 2: A Set with No Maximum

Problem: Find sup and inf of S = {1 - 1/n : n ∈ ℕ}

Step 1: List some elements

S = {0, 1/2, 2/3, 3/4, 4/5, ...}

Step 2: Find supremum

All elements are less than 1. As n increases, 1 - 1/n gets closer to 1. So sup(S) = 1.

Note: 1 is not in S (there's no n such that 1 - 1/n = 1).

Step 3: Find infimum

The smallest element is 0 (when n = 1). So inf(S) = 0 = min(S).

📋 How to Solve Most Questions

Step-by-Step: Finding Supremum and Infimum

  1. Understand the set — What are the elements? Is it bounded?
  2. Find upper/lower bounds — What numbers bound the set?
  3. Find the least/greatest bound — What's the tightest bound?
  4. Check if it's in the set — Is sup/inf actually an element?
  5. Verify your answer — Is it really the least/greatest?

The Completeness Axiom

Chapter 1: The Real Numbers Lesson 2 of 5 ⏱️ 30-40 minutes

🎯 Goal

Understand the completeness axiom and why it distinguishes the real numbers from the rationals.

In one sentence: The completeness axiom states that every non-empty set of real numbers that is bounded above has a least upper bound.

💡 The Idea

What is Completeness?

The completeness axiom is what makes the real numbers "complete" — there are no gaps. Every bounded set has a precise boundary.

Formally: If S is a non-empty subset of ℝ that is bounded above, then S has a least upper bound (supremum) in ℝ.

Why Rationals Aren't Complete

Consider the set S = {x ∈ ℚ : x² < 2}. This set is bounded above (by 2, for example), but it has no least upper bound in ℚ.

Why? Because √2 is irrational! The rationals have a "gap" at √2.

Geometric Interpretation

Think of the number line. The completeness axiom says there are no holes — every point that "should" exist actually does exist in ℝ.

The rationals have infinitely many holes (at every irrational number), but the reals fill them all in.

Why This Matters

Completeness is the foundation for all of analysis. Without it, we couldn't prove the Intermediate Value Theorem, the Extreme Value Theorem, or even guarantee that sequences converge. It's what makes calculus work.

📝 Example

Example 1: Finding a Supremum

Problem: Find the supremum of S = {x ∈ ℝ : x < 3}

Step 1: Check if S is bounded above

Yes, any number ≥ 3 is an upper bound. For example, 3, 4, 5 are all upper bounds.

Step 2: Find the least upper bound

3 is an upper bound. Is it the least?

Suppose M < 3 is an upper bound. Then (M + 3)/2 < 3, so (M + 3)/2 ∈ S. But (M + 3)/2 > M, contradicting that M is an upper bound.

Step 3: Conclusion

sup(S) = 3

Example 2: A Set Without a Maximum

Problem: Find the supremum of S = {1, 1/2, 1/3, 1/4, ...}

Step 1: Check if S is bounded above

Yes, 1 is an upper bound (all elements are ≤ 1).

Step 2: Find the least upper bound

1 ∈ S, so 1 is the maximum. Therefore sup(S) = 1.

Note: This set has a maximum, so the supremum equals the maximum.

📋 How to Solve Most Questions

Step-by-Step: Using the Completeness Axiom

  1. Identify the set S — What are the elements?
  2. Check if S is bounded above — Is there an upper bound?
  3. Apply the completeness axiom — S has a supremum
  4. Find the supremum — What is the least upper bound?
  5. Verify your answer — Is it really the least?

The Axioms

Chapter 1: The Real Numbers Lesson 1 of 5 ⏱️ 30-40 minutes

🎯 Goal

Understand the fundamental axioms that define the real number system.

In one sentence: The real numbers are defined by a set of axioms that describe their algebraic and order properties.

💡 The Idea

What Are Axioms?

Axioms are statements we assume to be true without proof. They form the foundation upon which we build everything else.

In real analysis, we start with axioms for the real numbers and prove everything from there.

Field Axioms

The real numbers satisfy these algebraic properties:

Addition Axioms

  • Closure: a + b is a real number
  • Commutativity: a + b = b + a
  • Associativity: (a + b) + c = a + (b + c)
  • Identity: There exists 0 such that a + 0 = a
  • Inverse: For every a, there exists -a such that a + (-a) = 0

Multiplication Axioms

  • Closure: a · b is a real number
  • Commutativity: a · b = b · a
  • Associativity: (a · b) · c = a · (b · c)
  • Identity: There exists 1 such that a · 1 = a
  • Inverse: For every a ≠ 0, there exists 1/a such that a · (1/a) = 1

Order Axioms

The real numbers have an ordering (we can compare them):

Order Properties

  • Trichotomy: For any a, b: exactly one of a < b, a = b, or a > b is true
  • Transitivity: If a < b and b < c, then a < c
  • Addition: If a < b, then a + c < b + c
  • Multiplication: If a < b and c > 0, then ac < bc

Why This Matters

Understanding axioms is crucial because everything in real analysis is built on them. When you prove a theorem, you're showing it follows logically from these basic assumptions.

📝 Example

Example 1: Prove that 0 · a = 0

Step 1: Start with what we know

We know 0 + 0 = 0 (additive identity)

Step 2: Multiply both sides by a

a · (0 + 0) = a · 0

Step 3: Use distributivity

a · 0 + a · 0 = a · 0

Step 4: Add -(a · 0) to both sides

a · 0 + a · 0 + (-(a · 0)) = a · 0 + (-(a · 0))
a · 0 + 0 = 0
a · 0 = 0

📋 How to Solve Most Questions

Step-by-Step: Proving Properties from Axioms

  1. State what you want to prove — Be clear about the goal
  2. List the axioms you'll use — Which ones apply?
  3. Start with known facts — Use axioms or previously proven results
  4. Apply axioms step by step — Each step must follow logically
  5. Reach your conclusion — Show the desired result

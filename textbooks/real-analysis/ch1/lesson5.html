The Archimedean Property

Chapter 1: The Real Numbers Lesson 5 of 5 ⏱️ 30-40 minutes

🎯 Goal

Understand the Archimedean property and how it ensures that the natural numbers are unbounded in the reals.

In one sentence: For any real number, there exists a natural number larger than it — the natural numbers are not bounded above.

💡 The Idea

What is the Archimedean Property?

The Archimedean property states that for any real number x, there exists a natural number n such that n > x.

In other words, ℕ is not bounded above in ℝ.

Equivalent Formulations

The Archimedean property is equivalent to:

  • For any ε > 0, there exists n ∈ ℕ such that 1/n < ε
  • For any x ∈ ℝ, there exists n ∈ ℕ such that n > x
  • The set ℕ has no upper bound in ℝ

Why It's True

Proof by contradiction: Suppose ℕ is bounded above. By completeness, it has a supremum, say M = sup(ℕ).

Since M is the least upper bound, M - 1 is not an upper bound, so there exists n ∈ ℕ with n > M - 1.

But then n + 1 > M, contradicting that M is an upper bound. Therefore, ℕ is not bounded above.

Applications

The Archimedean property is used to:

  • Prove that 1/n → 0 as n → ∞
  • Show that rationals are dense in ℝ
  • Prove the existence of decimal expansions
  • Establish properties of limits and continuity

Why This Matters

The Archimedean property ensures that the real numbers behave "normally" — there are no infinitely large or infinitely small numbers. It's essential for limits, sequences, and the foundations of calculus.

📝 Example

Example 1: Finding n for a Given ε

Problem: Given ε = 0.001, find n ∈ ℕ such that 1/n < ε

Step 1: Set up the inequality

We need 1/n < 0.001

Step 2: Solve for n

1/n < 0.001 ⟹ n > 1/0.001 = 1000

Step 3: Choose n

We can choose n = 1001 (or any integer greater than 1000)

Step 4: Verify

1/1001 ≈ 0.000999 < 0.001 ✓

Example 2: Proving a Limit Using Archimedean Property

Problem: Prove that lim(n→∞) 1/n = 0

Step 1: Definition of limit

For any ε > 0, we need to find N such that for all n > N, |1/n - 0| < ε

Step 2: Use Archimedean property

By the Archimedean property, there exists N ∈ ℕ such that 1/N < ε

Step 3: Show it works

For n > N, we have 1/n < 1/N < ε

Step 4: Conclusion

Therefore, lim(n→∞) 1/n = 0

📋 How to Solve Most Questions

Step-by-Step: Using the Archimedean Property

  1. Identify what you need — Do you need n > x or 1/n < ε?
  2. Apply the property — Such an n exists
  3. Find a specific n — Solve the inequality
  4. Verify your choice — Check that it works
  5. Use it in your proof — Apply to the problem at hand

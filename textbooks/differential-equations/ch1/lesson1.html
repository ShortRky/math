Introduction to Differential Equations

Chapter 1: First-Order Differential Equations Lesson 1 of 6 ⏱️ 25-35 minutes

🎯 Goal

Understand what a differential equation is, its basic terminology, and why it's important.

In one sentence: A differential equation is an equation that relates a function to its derivatives.

💡 The Idea

What is a Differential Equation?

A differential equation is an equation that involves an unknown function and its derivatives. For example:

dy/dx = 2x
dy/dx + y = eˣ
d²y/dx² + 3dy/dx + 2y = 0

Key Terminology

  • Order: The highest derivative in the equation
  • Solution: A function that satisfies the equation
  • General solution: Contains all possible solutions (usually has arbitrary constants)
  • Particular solution: A specific solution (constants determined by initial conditions)

Types of Differential Equations

Ordinary Differential Equations (ODEs)

Involve functions of one variable

Example: dy/dx = 3x²

Partial Differential Equations (PDEs)

Involve functions of multiple variables and partial derivatives

Example: ∂u/∂t = k·∂²u/∂x² (heat equation)

Why This Matters

Differential equations describe how things change — population growth, radioactive decay, heat flow, motion, and much more. They're the language of science and engineering.

📝 Example

Example 1: Verify a solution

Question: Is y = x² a solution to dy/dx = 2x?

Step 1: Find the derivative

y = x²
dy/dx = 2x

Step 2: Check if it satisfies the equation

The equation says dy/dx = 2x

We found dy/dx = 2x ✓

Step 3: Conclusion

Yes, y = x² is a solution.

Example 2: Find the order

Question: What is the order of d³y/dx³ + 2d²y/dx² - dy/dx + y = 0?

Step 1: Identify the highest derivative

The highest derivative is d³y/dx³ (third derivative)

Step 2: Conclusion

This is a third-order differential equation.

📋 How to Solve Most Questions

Step-by-Step: Verifying a Solution

  1. Take the derivative(s) — Find all derivatives needed
  2. Substitute into the equation — Replace y and its derivatives
  3. Simplify both sides — Make them look the same
  4. Check if they match — If yes, it's a solution
  5. State your conclusion — "Yes, it's a solution" or "No, it's not"

Step-by-Step: Identifying Order and Type

  1. Find the highest derivative — That's the order
  2. Check the variables — One variable = ODE, multiple = PDE
  3. Check if it's linear — Can it be written as a linear combination?
  4. Note any special forms — Separable, exact, etc.

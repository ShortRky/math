Exact Equations

Chapter 1: First-Order Differential Equations Lesson 4 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn how to identify and solve exact differential equations.

In one sentence: An exact equation is one where the left side is the total derivative of some function F(x, y).

💡 The Idea

What is an Exact Equation?

An equation of the form:

M(x, y)dx + N(x, y)dy = 0

is exact if there exists a function F(x, y) such that:

∂F/∂x = M and ∂F/∂y = N

Test for Exactness

The equation is exact if and only if:

∂M/∂y = ∂N/∂x

How to Solve

  1. Check if ∂M/∂y = ∂N/∂x
  2. Find F by integrating M with respect to x
  3. Differentiate F with respect to y and compare with N
  4. Find the missing terms
  5. Write the solution as F(x, y) = C

Why This Matters

Exact equations arise naturally in physics and engineering, especially in thermodynamics and conservation laws. They provide a systematic way to find potential functions.

📝 Example

Example 1: Solve (2xy + 3)dx + (x² + 4y)dy = 0

Step 1: Identify M and N

M = 2xy + 3, N = x² + 4y

Step 2: Check exactness

∂M/∂y = 2x, ∂N/∂x = 2x ✓

Step 3: Find F by integrating M

F = ∫M dx = ∫(2xy + 3)dx = x²y + 3x + h(y)

Step 4: Find h(y)

∂F/∂y = x² + h'(y) = N = x² + 4y
h'(y) = 4y
h(y) = 2y²

Step 5: Write solution

F(x, y) = x²y + 3x + 2y² = C

📋 How to Solve Most Questions

Step-by-Step: Solving Exact Equations

  1. Write in standard form — M dx + N dy = 0
  2. Check exactness — ∂M/∂y = ∂N/∂x?
  3. Integrate M with respect to x — Get F + h(y)
  4. Differentiate with respect to y — Compare with N
  5. Find h(y) — The missing function
  6. Write solution — F(x, y) = C

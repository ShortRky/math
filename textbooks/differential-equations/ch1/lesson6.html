Modeling with Differential Equations

Chapter 1: First-Order Differential Equations Lesson 6 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to translate real-world problems into differential equations and interpret their solutions.

In one sentence: Differential equations model how quantities change over time, from population growth to radioactive decay.

💡 The Idea

The Modeling Process

  1. Identify the changing quantity — What's changing?
  2. Determine the rate of change — How fast is it changing?
  3. Write the equation — Rate = function of current state
  4. Solve the equation — Find the function
  5. Interpret the solution — What does it mean?

Common Models

Population Growth

dP/dt = kP

Solution: P(t) = P₀e^(kt)

Growth is proportional to current population

Radioactive Decay

dN/dt = -λN

Solution: N(t) = N₀e^(-λt)

Decay is proportional to current amount

Newton's Law of Cooling

dT/dt = k(T - Tₐ)

Solution: T(t) = Tₐ + (T₀ - Tₐ)e^(kt)

Cooling rate is proportional to temperature difference

Why This Matters

Modeling is the heart of applied mathematics. Differential equations let us predict the future, understand the past, and make informed decisions in science, engineering, and economics.

📝 Example

Example 1: Population Growth

Problem: A population doubles every 10 years. If the initial population is 1000, find the population after 25 years.

Step 1: Set up the equation

dP/dt = kP

Step 2: Solve

P(t) = P₀e^(kt)

Step 3: Find k using doubling time

2P₀ = P₀e^(10k)
2 = e^(10k)
k = ln(2)/10 ≈ 0.0693

Step 4: Find P(25)

P(25) = 1000e^(0.0693 × 25) ≈ 1000e^(1.7325) ≈ 5657

Example 2: Radioactive Decay

Problem: A radioactive substance has a half-life of 5 years. If you start with 100g, how much remains after 12 years?

Step 1: Set up the equation

dN/dt = -λN

Step 2: Solve

N(t) = N₀e^(-λt)

Step 3: Find λ using half-life

N₀/2 = N₀e^(-5λ)
1/2 = e^(-5λ)
λ = ln(2)/5 ≈ 0.1386

Step 4: Find N(12)

N(12) = 100e^(-0.1386 × 12) ≈ 100e^(-1.6632) ≈ 19g

📋 How to Solve Most Questions

Step-by-Step: Modeling Problems

  1. Read carefully — What's changing? What's the rate?
  2. Define variables — Let P = population, t = time, etc.
  3. Write the equation — dP/dt = ...
  4. Solve the equation — Use appropriate method
  5. Use initial conditions — Find the constant
  6. Answer the question — Plug in the given values

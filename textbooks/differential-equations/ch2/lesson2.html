Characteristic Equations

Chapter 2: Second-Order Linear Equations Lesson 2 of 6 ⏱️ 35-45 minutes

🎯 Goal

Master the characteristic equation method for solving second-order linear homogeneous differential equations.

In one sentence: The characteristic equation transforms a differential equation into an algebraic equation that's easier to solve.

💡 The Idea

Why It Works

For the equation ay'' + by' + cy = 0, we guess a solution of the form y = e^(rt). Substituting:

ar²e^(rt) + bre^(rt) + ce^(rt) = 0
e^(rt)(ar² + br + c) = 0

Since e^(rt) ≠ 0, we need ar² + br + c = 0.

The Quadratic Formula

The roots of ar² + br + c = 0 are:

r = (-b ± √(b² - 4ac)) / (2a)

The discriminant Δ = b² - 4ac determines the type of roots.

Three Cases

  • Δ > 0: Two distinct real roots → y = C₁e^(r₁t) + C₂e^(r₂t)
  • Δ = 0: One repeated root → y = C₁e^(rt) + C₂te^(rt)
  • Δ < 0: Complex roots r = α ± βi → y = e^(αt)(C₁cos(βt) + C₂sin(βt))

Why This Matters

The characteristic equation is the key tool for solving homogeneous equations. Understanding how to find and interpret roots is essential for all second-order equations.

📝 Example

Example 1: Solve y'' + 2y' - 8y = 0

Step 1: Write characteristic equation

r² + 2r - 8 = 0

Step 2: Solve using quadratic formula

r = (-2 ± √(4 + 32)) / 2 = (-2 ± √36) / 2 = (-2 ± 6) / 2
r₁ = 2, r₂ = -4

Step 3: Write solution

y = C₁e^(2t) + C₂e^(-4t)

Example 2: Solve y'' + 4y' + 13y = 0

Step 1: Write characteristic equation

r² + 4r + 13 = 0

Step 2: Solve using quadratic formula

r = (-4 ± √(16 - 52)) / 2 = (-4 ± √(-36)) / 2 = (-4 ± 6i) / 2
r = -2 ± 3i

Step 3: Write solution

y = e^(-2t)(C₁cos(3t) + C₂sin(3t))

📋 How to Solve Most Questions

Step-by-Step: Using the Characteristic Equation

  1. Write the equation — ay'' + by' + cy = 0
  2. Form the characteristic equation — ar² + br + c = 0
  3. Calculate the discriminant — Δ = b² - 4ac
  4. Find the roots — Use factoring or quadratic formula
  5. Determine the type — Real distinct, repeated, or complex?
  6. Write the solution — Use the appropriate form

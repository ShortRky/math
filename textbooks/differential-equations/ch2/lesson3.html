Non-Homogeneous Equations

Chapter 2: Second-Order Linear Equations Lesson 3 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to solve second-order linear non-homogeneous differential equations.

In one sentence: The general solution to a non-homogeneous equation is the homogeneous solution plus a particular solution.

💡 The Idea

Standard Form

A second-order linear non-homogeneous equation has the form:

ay'' + by' + cy = f(t)

Where f(t) is the forcing function (not zero).

Structure of the Solution

The general solution is:

y(t) = y_h(t) + y_p(t)

Where:

  • y_h(t) is the homogeneous solution (complementary solution)
  • y_p(t) is a particular solution

Finding y_h

Solve the homogeneous equation ay'' + by' + cy = 0 using the characteristic equation method.

Finding y_p

Two main methods:

  • Undetermined Coefficients — Guess a form based on f(t)
  • Variation of Parameters — More general, works for any f(t)

Why This Matters

Non-homogeneous equations model real-world systems with external forces — springs with driving forces, circuits with voltage sources, and more. They're essential for engineering and physics.

📝 Example

Example 1: Solve y'' - 3y' + 2y = 4

Step 1: Find y_h

r² - 3r + 2 = 0
(r - 1)(r - 2) = 0
y_h = C₁e^t + C₂e^(2t)

Step 2: Find y_p

Since f(t) = 4 (constant), guess y_p = A

y_p' = 0, y_p'' = 0
0 - 0 + 2A = 4
A = 2
y_p = 2

Step 3: Write general solution

y = C₁e^t + C₂e^(2t) + 2

📋 How to Solve Most Questions

Step-by-Step: Solving Non-Homogeneous Equations

  1. Find y_h — Solve the homogeneous equation
  2. Find y_p — Use undetermined coefficients or variation of parameters
  3. Write general solution — y = y_h + y_p
  4. Apply initial conditions — If given, find C₁ and C₂

Homogeneous Equations

Chapter 2: Second-Order Linear Equations Lesson 1 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn how to solve second-order linear homogeneous differential equations with constant coefficients.

In one sentence: Homogeneous equations have the form ay'' + by' + cy = 0 and are solved using the characteristic equation.

💡 The Idea

Standard Form

A second-order linear homogeneous equation has the form:

ay'' + by' + cy = 0

Where a, b, and c are constants and a ≠ 0.

The Characteristic Equation

Assume y = e^(rt). Substituting into the equation gives:

ar² + br + c = 0

This is the characteristic equation.

Types of Solutions

Two Distinct Real Roots (r₁ ≠ r₂)

y = C₁e^(r₁t) + C₂e^(r₂t)

Repeated Real Root (r₁ = r₂ = r)

y = C₁e^(rt) + C₂te^(rt)

Complex Roots (r = α ± βi)

y = e^(αt)(C₁cos(βt) + C₂sin(βt))

Why This Matters

Second-order equations model many physical systems — springs, circuits, pendulums. Understanding homogeneous solutions is the foundation for solving more complex non-homogeneous equations.

📝 Example

Example 1: Solve y'' - 5y' + 6y = 0

Step 1: Write the characteristic equation

r² - 5r + 6 = 0

Step 2: Solve for r

(r - 2)(r - 3) = 0
r₁ = 2, r₂ = 3

Step 3: Write the solution

y = C₁e^(2t) + C₂e^(3t)

Example 2: Solve y'' + 4y' + 4y = 0

Step 1: Write the characteristic equation

r² + 4r + 4 = 0

Step 2: Solve for r

(r + 2)² = 0
r = -2 (repeated)

Step 3: Write the solution

y = C₁e^(-2t) + C₂te^(-2t)

📋 How to Solve Most Questions

Step-by-Step: Solving Homogeneous Equations

  1. Write in standard form — ay'' + by' + cy = 0
  2. Write characteristic equation — ar² + br + c = 0
  3. Solve for r — Use factoring or quadratic formula
  4. Determine the type — Distinct, repeated, or complex?
  5. Write the solution — Use the appropriate form

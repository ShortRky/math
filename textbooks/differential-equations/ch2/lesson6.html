Applications: Springs and Circuits

Chapter 2: Second-Order Linear Equations Lesson 6 of 6 ⏱️ 45-55 minutes

🎯 Goal

Apply second-order differential equations to model and solve real-world problems with springs and electrical circuits.

In one sentence: Springs and circuits are modeled by second-order equations, and their behavior depends on damping and forcing.

💡 The Idea

Mass-Spring System

A mass m attached to a spring with constant k and damping c:

mx'' + cx' + kx = F(t)

Where:

  • x is displacement from equilibrium
  • c is the damping coefficient
  • F(t) is the external force

RLC Circuit

A circuit with resistance R, inductance L, and capacitance C:

Lq'' + Rq' + (1/C)q = E(t)

Where:

  • q is the charge on the capacitor
  • E(t) is the voltage source

Types of Motion

Undamped (c = 0)

Pure oscillation: x = Acos(ωt + φ)

Underdamped (c² < 4mk)

Oscillation with decay: x = e^(-αt)(Acos(βt) + Bsin(βt))

Critically Damped (c² = 4mk)

Fastest return without oscillation: x = (C₁ + C₂t)e^(-αt)

Overdamped (c² > 4mk)

Slow return without oscillation: x = C₁e^(r₁t) + C₂e^(r₂t)

Why This Matters

Understanding these applications is crucial for engineering. Springs model mechanical systems, while circuits model electronic devices. The same math applies to both!

📝 Example

Example 1: Mass-Spring System

Problem: A 2 kg mass is attached to a spring with k = 8 N/m and c = 4 N·s/m. Find the motion if x(0) = 1 and x'(0) = 0.

Step 1: Set up the equation

2x'' + 4x' + 8x = 0
x'' + 2x' + 4x = 0

Step 2: Solve the characteristic equation

r² + 2r + 4 = 0
r = (-2 ± √(4 - 16))/2 = (-2 ± √(-12))/2 = -1 ± i√3

Step 3: Write the general solution

x = e^(-t)(C₁cos(√3t) + C₂sin(√3t))

Step 4: Apply initial conditions

x(0) = 1: C₁ = 1
x'(0) = 0: -C₁ + √3C₂ = 0 → C₂ = 1/√3
x = e^(-t)(cos(√3t) + (1/√3)sin(√3t))

This is underdamped motion — oscillation with exponential decay.

📋 How to Solve Most Questions

Step-by-Step: Solving Application Problems

  1. Identify the system — Spring or circuit?
  2. Write the equation — Use the appropriate form
  3. Find the type — Underdamped, critically damped, or overdamped?
  4. Solve the equation — Use characteristic equation method
  5. Apply initial conditions — Find the constants
  6. Interpret the result — What does the motion look like?

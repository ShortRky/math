Undetermined Coefficients

Chapter 2: Second-Order Linear Equations Lesson 4 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn the method of undetermined coefficients to find particular solutions to non-homogeneous equations.

In one sentence: Undetermined coefficients works by guessing a solution form similar to f(t) and solving for the constants.

💡 The Idea

When to Use

Use undetermined coefficients when f(t) is:

  • A polynomial
  • An exponential function
  • A sine or cosine
  • A combination of these

The Method

  1. Find y_h (homogeneous solution)
  2. Guess the form of y_p based on f(t)
  3. If any term in the guess is in y_h, multiply by t (or t² if needed)
  4. Substitute the guess into the equation
  5. Solve for the coefficients

Common Guesses

f(t) = polynomial

Guess: y_p = At^n + At^(n-1) + ... + A₀

f(t) = e^(kt)

Guess: y_p = Ae^(kt)

f(t) = sin(kt) or cos(kt)

Guess: y_p = Acos(kt) + Bsin(kt)

Why This Matters

Undetermined coefficients is a systematic, algebraic method that works for many common forcing functions. It's faster than variation of parameters for these cases.

📝 Example

Example 1: Solve y'' - 3y' + 2y = 4e^(3t)

Step 1: Find y_h

r² - 3r + 2 = 0
(r - 1)(r - 2) = 0
y_h = C₁e^t + C₂e^(2t)

Step 2: Guess y_p

Since f(t) = 4e^(3t), guess y_p = Ae^(3t)

Step 3: Substitute and solve

y_p = Ae^(3t)
y_p' = 3Ae^(3t)
y_p'' = 9Ae^(3t)
9Ae^(3t) - 9Ae^(3t) + 2Ae^(3t) = 4e^(3t)
2Ae^(3t) = 4e^(3t)
A = 2
y_p = 2e^(3t)

Step 4: Write general solution

y = C₁e^t + C₂e^(2t) + 2e^(3t)

📋 How to Solve Most Questions

Step-by-Step: Undetermined Coefficients

  1. Find y_h — Solve the homogeneous equation
  2. Choose the form — Based on f(t), guess y_p
  3. Check for duplication — If guess terms are in y_h, multiply by t
  4. Substitute — Plug y_p into the equation
  5. Solve for coefficients — Match terms on both sides
  6. Write general solution — y = y_h + y_p

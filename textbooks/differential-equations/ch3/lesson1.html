Introduction to Systems

Chapter 3: Systems of Differential Equations Lesson 1 of 5 ⏱️ 35-45 minutes

🎯 Goal

Understand what systems of differential equations are and how to write them in matrix form.

In one sentence: Systems of differential equations involve multiple unknown functions that are related to each other.

💡 The Idea

What is a System?

A system of differential equations involves multiple unknown functions. For example:

dx/dt = 3x + 2y
dy/dt = x - y

Here x(t) and y(t) are both unknown functions of t.

Matrix Form

We can write systems using matrices. Let X = [x, y]ᵀ:

dX/dt = AX

Where A is the coefficient matrix:

A = [[3, 2], [1, -1]]

Why Systems?

  • Multiple related quantities — Predator-prey models, chemical reactions
  • Higher-order equations — Can be converted to systems
  • More realistic models — Real systems have multiple variables

Converting Higher-Order to Systems

For y'' + ay' + by = 0, let x₁ = y and x₂ = y':

x₁' = x₂
x₂' = -bx₁ - ax₂

Why This Matters

Systems of equations model complex interactions in biology, chemistry, physics, and economics. They're essential for understanding how multiple quantities change together.

📝 Example

Example 1: Write in Matrix Form

Problem: Write the system in matrix form:

dx/dt = 2x - y
dy/dt = x + 3y

Step 1: Define X

X = [x, y]ᵀ

Step 2: Find matrix A

A = [[2, -1], [1, 3]]

Step 3: Write the system

dX/dt = [[2, -1], [1, 3]]X

Example 2: Convert to System

Problem: Convert y'' + 2y' + 5y = 0 to a system

Step 1: Define variables

x₁ = y
x₂ = y'

Step 2: Write the system

x₁' = x₂
x₂' = -5x₁ - 2x₂

Step 3: Matrix form

dX/dt = [[0, 1], [-5, -2]]X

📋 How to Solve Most Questions

Step-by-Step: Working with Systems

  1. Identify the unknowns — What are x(t), y(t), etc.?
  2. Write the equations — dx/dt = ..., dy/dt = ...
  3. Define X — X = [x, y, ...]ᵀ
  4. Find matrix A — The coefficient matrix
  5. Write in matrix form — dX/dt = AX

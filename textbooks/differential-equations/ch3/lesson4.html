Phase Portraits

Chapter 3: Systems of Differential Equations Lesson 4 of 5 ⏱️ 40-50 minutes

🎯 Goal

Learn how to visualize the behavior of systems of differential equations using phase portraits.

In one sentence: Phase portraits show how solutions move through the phase plane, revealing stability and long-term behavior.

💡 The Idea

What is a Phase Portrait?

A phase portrait is a plot of solution curves in the phase plane (x-y plane). Each curve represents a solution, showing how x and y change together over time.

Types of Equilibrium Points

Node

Both eigenvalues are real and have the same sign

Stable node: both negative (solutions approach origin)

Unstable node: both positive (solutions move away)

Saddle Point

Eigenvalues are real with opposite signs

Solutions approach along one direction, move away along another

Spiral

Complex eigenvalues

Stable spiral: negative real part (spiral inward)

Unstable spiral: positive real part (spiral outward)

Center

Purely imaginary eigenvalues

Solutions form closed orbits (circles or ellipses)

Determining Type from Eigenvalues

  • Real, same sign: Node
  • Real, opposite signs: Saddle
  • Complex, nonzero real part: Spiral
  • Purely imaginary: Center

Why This Matters

Phase portraits give you a visual understanding of system behavior without solving the equations completely. They're essential for understanding stability in engineering and science.

📝 Example

Example 1: Classify the equilibrium

Problem: Classify the equilibrium at the origin for dX/dt = [[-2, 1], [1, -2]]X

Step 1: Find eigenvalues

det(A - λI) = (-2-λ)² - 1 = 0
λ² + 4λ + 3 = 0
(λ + 1)(λ + 3) = 0
λ₁ = -1, λ₂ = -3

Step 2: Classify

Both eigenvalues are real and negative → Stable node

Step 3: Interpret

All solutions approach the origin as t → ∞

Example 2: Complex eigenvalues

Problem: Classify the equilibrium for dX/dt = [[0, 1], [-1, 0]]X

Step 1: Find eigenvalues

det(A - λI) = -λ( -λ) - (-1) = 0
λ² + 1 = 0
λ = ±i

Step 2: Classify

Purely imaginary eigenvalues → Center

Step 3: Interpret

Solutions form closed orbits around the origin

📋 How to Solve Most Questions

Step-by-Step: Analyzing Phase Portraits

  1. Find eigenvalues — Solve det(A - λI) = 0
  2. Determine the type — Real? Complex? Signs?
  3. Classify the equilibrium — Node, saddle, spiral, or center?
  4. Determine stability — Stable, unstable, or neutral?
  5. Sketch the portrait — Draw representative solution curves

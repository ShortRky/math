Applications of Integration

Chapter 5: Integration Lesson 7 of 7 ⏱️ 40-50 minutes

🎯 Goal

Learn how to use integration to solve real-world problems.

In one sentence: Integration is used to find volumes, work, and other quantities that accumulate over an interval.

💡 The Idea

Common Applications

Volumes of Revolution

Rotate a region around an axis to create a 3D shape

V = π∫ₐᵇ [f(x)]²dx (disk method)

Work

Work = ∫ₐᵇ F(x)dx

Force times distance, when force varies

Average Value

avg = (1/(b-a))∫ₐᵇ f(x)dx

Why This Matters

Integration is used everywhere — physics, engineering, economics, biology, and more. It's one of the most practical tools in mathematics.

📝 Example

Example: Find volume of y = √x rotated around x-axis from 0 to 4

Step 1: Set up integral

V = π∫₀⁴ [√x]²dx = π∫₀⁴ xdx

Step 2: Evaluate

= π[x²/2]₀⁴ = π(16/2 - 0) = 8π

📋 How to Solve Most Questions

Step-by-Step: Applications of Integration

  1. Identify the application — Volume, work, average?
  2. Draw a diagram — Visualize the situation
  3. Set up the integral — Use the appropriate formula
  4. Evaluate — Find the antiderivative
  5. Interpret — What does the answer mean?

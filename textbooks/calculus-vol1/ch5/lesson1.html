Antiderivatives

Chapter 5: Integration Lesson 1 of 7 ⏱️ 30-40 minutes

🎯 Goal

Understand what an antiderivative is and learn basic antiderivative formulas.

In one sentence: An antiderivative is the reverse of a derivative — finding a function whose derivative is the given function.

💡 The Idea

What Is an Antiderivative?

If F'(x) = f(x), then F(x) is an antiderivative of f(x).

For example, since d/dx [x²] = 2x, then x² is an antiderivative of 2x.

Notation

∫f(x)dx = F(x) + C

The integral sign ∫ means "find the antiderivative"

C is the constant of integration

Basic Antiderivatives

Power Rule

∫xⁿdx = xⁿ⁺¹/(n+1) + C (n ≠ -1)

Other Basic Forms

∫dx = x + C

∫cos(x)dx = sin(x) + C

∫sin(x)dx = -cos(x) + C

∫eˣdx = eˣ + C

Why This Matters

Antiderivatives are the foundation of integration, which is used to find areas, volumes, and solve many real-world problems.

📝 Example

Example: Find ∫x³dx

Step 1: Identify n

n = 3

Step 2: Apply power rule

∫x³dx = x³⁺¹/(3+1) + C = x⁴/4 + C

Step 3: Check

d/dx [x⁴/4 + C] = 4x³/4 = x³ ✓

📋 How to Solve Most Questions

Step-by-Step: Finding Antiderivatives

  1. Identify the form — What type of function?
  2. Apply the appropriate rule — Power, trig, exponential
  3. Add +C — Don't forget the constant
  4. Check — Differentiate to verify

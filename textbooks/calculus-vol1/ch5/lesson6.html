Area Between Curves

Chapter 5: Integration Lesson 6 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn how to find the area between two curves.

In one sentence: The area between two curves is the integral of the top curve minus the bottom curve.

💡 The Idea

Area Between Curves

If f(x) ≥ g(x) on [a, b], then the area between them is:

A = ∫ₐᵇ [f(x) - g(x)]dx

Finding Intersection Points

To find where curves intersect, set f(x) = g(x) and solve for x.

Why This Matters

Finding areas between curves is used in physics, engineering, and economics to find quantities like work, consumer surplus, and more.

📝 Example

Example: Find area between y = x² and y = x on [0, 1]

Step 1: Identify top and bottom

On [0, 1]: x ≥ x², so top = x, bottom = x²

Step 2: Set up integral

A = ∫₀¹ [x - x²]dx

Step 3: Evaluate

= [x²/2 - x³/3]₀¹ = (1/2 - 1/3) - 0 = 1/6

📋 How to Solve Most Questions

Step-by-Step: Finding Area Between Curves

  1. Graph the curves — Visualize the region
  2. Find intersection points — Set f(x) = g(x)
  3. Identify top and bottom — Which is greater?
  4. Set up integral — ∫[top - bottom]dx
  5. Evaluate — Find the area

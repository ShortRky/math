Maxima and Minima

Chapter 4: Applications of Derivatives Lesson 3 of 6 ⏱️ 35-45 minutes

🎯 Goal

Learn how to find the highest and lowest values of a function.

In one sentence: Maxima and minima are the highest and lowest points on a graph, found by setting the derivative equal to zero.

💡 The Idea

Critical Points

A critical point is where f'(x) = 0 or f'(x) is undefined. Maxima and minima occur at critical points.

First Derivative Test

If f'(x) changes from positive to negative at c, then f(c) is a local maximum.

If f'(x) changes from negative to positive at c, then f(c) is a local minimum.

Second Derivative Test

If f'(c) = 0 and f''(c) > 0, then f(c) is a local minimum.

If f'(c) = 0 and f''(c) < 0, then f(c) is a local maximum.

Why This Matters

Finding maxima and minima is essential for optimization — maximizing profit, minimizing cost, finding best designs.

📝 Example

Example: Find local extrema of f(x) = x³ - 3x

Step 1: Find f'(x)

f'(x) = 3x² - 3 = 3(x² - 1) = 3(x-1)(x+1)

Step 2: Find critical points

f'(x) = 0 when x = 1 or x = -1

Step 3: Test intervals

x < -1: f'(x) > 0 (increasing)

-1 < x < 1: f'(x) < 0 (decreasing)

x > 1: f'(x) > 0 (increasing)

Step 4: Conclusion

Local max at x = -1, local min at x = 1

📋 How to Solve Most Questions

Step-by-Step: Finding Maxima and Minima

  1. Find f'(x) — Differentiate the function
  2. Find critical points — Set f'(x) = 0
  3. Test intervals — Check sign of f'(x) around critical points
  4. Apply first derivative test — Sign change indicates max or min
  5. Or use second derivative test — Check f''(c)

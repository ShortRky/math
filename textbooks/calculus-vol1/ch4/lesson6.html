Optimization Problems

Chapter 4: Applications of Derivatives Lesson 6 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to use calculus to find maximum or minimum values in real-world problems.

In one sentence: Optimization uses derivatives to find the best possible solution to a problem.

💡 The Idea

What Is Optimization?

Optimization means finding the maximum or minimum value of a quantity, subject to constraints.

Examples:

  • Maximize area with fixed perimeter
  • Minimize cost of materials
  • Maximize profit
  • Minimize travel time

The Process

  1. Draw a diagram
  2. Define variables
  3. Write an equation for what you're optimizing
  4. Use constraints to reduce to one variable
  5. Find critical points
  6. Check for maximum or minimum

Why This Matters

Optimization is used everywhere — engineering design, business decisions, resource allocation, and scientific research.

📝 Example

Example: Find the rectangle with maximum area and perimeter 20

Step 1: Define variables

Let x = length, y = width

Step 2: Write equations

Perimeter: 2x + 2y = 20, so y = 10 - x

Area: A = x·y = x(10-x) = 10x - x²

Step 3: Find critical points

A'(x) = 10 - 2x = 0, so x = 5

Step 4: Check

A''(x) = -2 < 0, so x = 5 is a maximum

Maximum area = 5·5 = 25

📋 How to Solve Most Questions

Step-by-Step: Solving Optimization Problems

  1. Read carefully — What are you optimizing?
  2. Draw a diagram — Visualize the situation
  3. Define variables — What can change?
  4. Write equations — Objective function and constraints
  5. Reduce to one variable — Use constraints
  6. Find critical points — Set derivative to zero
  7. Check — Is it a max or min?

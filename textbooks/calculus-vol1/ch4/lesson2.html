Linear Approximations

Chapter 4: Applications of Derivatives Lesson 2 of 6 ⏱️ 30-40 minutes

🎯 Goal

Learn how to use derivatives to approximate function values near a known point.

In one sentence: Linear approximation uses the tangent line to estimate function values near a point.

💡 The Idea

What Is Linear Approximation?

Near a point x = a, the function f(x) can be approximated by the tangent line:

f(x) ≈ f(a) + f'(a)(x - a)

This is called the linear approximation or tangent line approximation.

When Does It Work Well?

The approximation is good when x is close to a. The farther away you get, the worse the approximation.

Why This Matters

Linear approximations are used in science and engineering to simplify complex calculations and make quick estimates.

📝 Example

Example: Approximate √(4.1)

Step 1: Choose a and f

f(x) = √x, a = 4 (since √4 = 2 is easy)

Step 2: Find f'(a)

f'(x) = 1/(2√x), f'(4) = 1/4

Step 3: Apply formula

f(4.1) ≈ f(4) + f'(4)(4.1-4) = 2 + (1/4)(0.1) = 2.025

Actual value: √4.1 ≈ 2.0248...

📋 How to Solve Most Questions

Step-by-Step: Linear Approximation

  1. Choose a point a — Pick a value close to x where f(a) is known
  2. Find f(a) and f'(a) — Evaluate function and derivative
  3. Apply the formula — f(x) ≈ f(a) + f'(a)(x-a)
  4. Calculate — Plug in the numbers

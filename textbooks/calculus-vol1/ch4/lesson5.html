Curve Sketching

Chapter 4: Applications of Derivatives Lesson 5 of 6 ⏱️ 40-50 minutes

🎯 Goal

Learn how to use derivatives to sketch accurate graphs of functions.

In one sentence: Curve sketching uses first and second derivatives to understand where a function increases, decreases, and curves.

💡 The Idea

What Derivatives Tell Us

First Derivative

f'(x) > 0: function is increasing

f'(x) < 0: function is decreasing

f'(x) = 0: potential max or min

Second Derivative

f''(x) > 0: function is concave up

f''(x) < 0: function is concave down

f''(x) = 0: potential inflection point

Why This Matters

Curve sketching helps you visualize functions and understand their behavior without plotting hundreds of points.

📝 Example

Example: Sketch f(x) = x³ - 3x

Step 1: Find f'(x)

f'(x) = 3x² - 3 = 3(x-1)(x+1)

Step 2: Find critical points

f'(x) = 0 at x = -1, 1

Step 3: Find f''(x)

f''(x) = 6x

Step 4: Analyze

Increasing: x < -1 or x > 1

Decreasing: -1 < x < 1

Concave up: x > 0

Concave down: x < 0

📋 How to Solve Most Questions

Step-by-Step: Curve Sketching

  1. Find f'(x) — Differentiate
  2. Find critical points — Set f'(x) = 0
  3. Find f''(x) — Differentiate again
  4. Analyze intervals — Where increasing/decreasing?
  5. Find inflection points — Where concavity changes
  6. Sketch — Draw the curve

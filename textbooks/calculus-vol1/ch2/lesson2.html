Evaluating Limits Graphically

Chapter 2: Limits and Continuity Lesson 2 of 6 ⏱️ 25-35 minutes

🎯 Goal

Learn how to find limits by looking at graphs of functions.

In one sentence: To find a limit from a graph, trace the curve from both sides toward the point and see what y-value it approaches.

💡 The Idea

Reading Limits from Graphs

When you have a graph, you can find limits by:

  1. Start from the left side of the point
  2. Move toward the point along the curve
  3. Note what y-value you're approaching
  4. Repeat from the right side
  5. If both sides approach the same value, that's the limit

When Limits Don't Exist

A limit doesn't exist if:

  • Left and right limits are different
  • The function goes to infinity
  • The function oscillates wildly

Why This Matters

Graphs give us a visual way to understand limits, which helps build intuition before we learn algebraic techniques.

📝 Example

Example: Find lim(x→1) f(x) from graph

Step 1: Approach from left

As x → 1⁻, f(x) → 3

Step 2: Approach from right

As x → 1⁺, f(x) → 3

Step 3: Compare

Both sides approach 3, so lim(x→1) f(x) = 3

📋 How to Solve Most Questions

Step-by-Step: Finding Limits from Graphs

  1. Locate the point — Find x = a on the graph
  2. Trace from left — Follow the curve toward x = a from the left
  3. Note the y-value — What y-value are you approaching?
  4. Trace from right — Follow the curve toward x = a from the right
  5. Compare — If same, that's the limit. If different, limit doesn't exist

Introduction to Limits

Chapter 2: Limits and Continuity Lesson 1 of 6 ⏱️ 25-35 minutes

🎯 Goal

Understand what a limit is and how to evaluate it using tables and graphs.

In one sentence: A limit is the value a function approaches as the input gets closer to a certain point.

💡 The Idea

What is a Limit?

The limit of f(x) as x approaches a is the value f(x) gets closer to as x gets closer to a.

lim(x→a) f(x) = L

Read as "the limit of f(x) as x approaches a equals L"

Key Idea

The limit is about what happens near a point, not at the point. The function doesn't need to be defined at a.

Why This Matters

Limits are the foundation of calculus. They allow us to define derivatives and integrals rigorously.

📝 Example

Example: Find lim(x→2) (x² - 4)/(x - 2)

Step 1: Try direct substitution

f(2) = (4-4)/(2-2) = 0/0 (undefined)

Step 2: Simplify

(x² - 4)/(x - 2) = (x-2)(x+2)/(x-2) = x + 2

Step 3: Evaluate

lim(x→2) (x + 2) = 4

📋 How to Solve Most Questions

Step-by-Step: Evaluating Limits

  1. Try direct substitution — Plug in the value
  2. If undefined — Simplify, factor, or rationalize
  3. Try again — Substitute after simplifying
  4. Use a table — If still stuck, test values near the point
  5. State the answer — The limit is the value approached

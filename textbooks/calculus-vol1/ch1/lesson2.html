Domain and Range

Chapter 1: Functions and Graphs Lesson 2 of 5 ⏱️ 25-35 minutes

🎯 Goal

Learn how to identify the domain (all possible inputs) and range (all possible outputs) of a function.

In one sentence: Domain is what you CAN put in; range is what you CAN get out.

💡 The Idea

What is Domain?

The domain of a function is the set of all input values (x-values) for which the function is defined. It's the list of all numbers you're allowed to put into the function.

Example: f(x) = x²

What numbers can you square?

  • Can you square 2? Yes: 2² = 4 ✓
  • Can you square -3? Yes: (-3)² = 9 ✓
  • Can you square 0? Yes: 0² = 0 ✓
  • Can you square 1.5? Yes: (1.5)² = 2.25 ✓

Domain: All real numbers, written as (-∞, ∞) or ℝ

What is Range?

The range of a function is the set of all output values (y-values) that the function can produce. It's the list of all numbers you can get out of the function.

Example: f(x) = x²

What numbers can you get by squaring?

  • Can you get 4? Yes: 2² = 4 ✓
  • Can you get 9? Yes: 3² = 9 ✓
  • Can you get 0? Yes: 0² = 0 ✓
  • Can you get -4? No: no number squared gives a negative ✗

Range: All non-negative numbers, written as [0, ∞)

Common Restrictions on Domain

Some functions have inputs that aren't allowed. Watch for these common restrictions:

1. Division by Zero

You can't divide by zero!

f(x) = 1/x

x cannot be 0, so domain is all real numbers except 0: (-∞, 0) ∪ (0, ∞)

2. Square Roots of Negative Numbers

You can't take the square root of a negative number (in real numbers)!

f(x) = √x

x must be ≥ 0, so domain is [0, ∞)

3. Logarithms of Non-Positive Numbers

You can't take the log of zero or a negative number!

f(x) = ln(x)

x must be > 0, so domain is (0, ∞)

Why This Matters

Knowing the domain and range helps you understand what a function can and cannot do. In calculus, we'll need to know these to find limits, derivatives, and integrals correctly.

📝 Example

Find the domain and range of f(x) = √(x - 2)

Step 1: Find the domain

The expression under the square root must be ≥ 0:

x - 2 ≥ 0
x ≥ 2

Domain: [2, ∞)

Step 2: Find the range

When x = 2: f(2) = √0 = 0

As x increases, √(x-2) increases without bound

Range: [0, ∞)

Find the domain and range of g(x) = 1/(x + 3)

Step 1: Find the domain

The denominator cannot be zero:

x + 3 ≠ 0
x ≠ -3

Domain: (-∞, -3) ∪ (-3, ∞)

Step 2: Find the range

The output can be any number except 0 (since 1 divided by anything is never 0)

Range: (-∞, 0) ∪ (0, ∞)

📋 How to Solve Most Questions

Step-by-Step: Finding Domain

  1. Look for restrictions: Check for division, square roots, logarithms
  2. Set restrictions: Write what cannot happen (e.g., denominator ≠ 0)
  3. Solve inequalities: Find what values ARE allowed
  4. Write the domain: Use interval notation or set notation

Step-by-Step: Finding Range

  1. Find the minimum: What's the smallest output possible?
  2. Find the maximum: What's the largest output possible?
  3. Check for gaps: Are there any outputs that can't be achieved?
  4. Write the range: Use interval notation or set notation

What is a Function?

Chapter 1: Functions and Graphs Lesson 1 of 5 ⏱️ 20-30 minutes

🎯 Goal

Understand what a function is, how to identify one, and why functions are the foundation of calculus.

In one sentence: A function is a rule that takes an input and gives exactly one output.

💡 The Idea

What is a Function?

A function is a relationship between two sets where each input has exactly one output. Think of it like a machine: you put something in, and you get something out.

Example: A Vending Machine

A vending machine is a function:

  • Input: You press a button (like "A3")
  • Output: You get a specific snack
  • Rule: Each button gives exactly one type of snack

If pressing "A3" sometimes gave chips and sometimes gave candy, it wouldn't be a function!

Function Notation

Mathematicians use special notation to write functions. Instead of saying "the output when the input is x," we write:

f(x) = "f of x"

Where:

  • f is the name of the function
  • x is the input (also called the argument)
  • f(x) is the output

Example: f(x) = 2x + 3

This function takes any number, multiplies it by 2, and adds 3.

f(1) = 2(1) + 3 = 5
f(4) = 2(4) + 3 = 11
f(-2) = 2(-2) + 3 = -1

Why This Matters

Functions are the language of calculus. Every concept we'll learn — limits, derivatives, integrals — is built on understanding functions. Master this, and everything else becomes clearer.

📝 Example

Is this a function?

Consider the relationship: "The square of a number"

f(x) = x²

Check: Does every input have exactly one output?

  • Input 2 → Output 4 ✓
  • Input -3 → Output 9 ✓
  • Input 0 → Output 0 ✓

Yes! Every input gives exactly one output. This is a function.

Is this a function?

Consider: "The square root of a number"

g(x) = √x

Check: Does every input have exactly one output?

  • Input 4 → Output 2 ✓
  • Input 9 → Output 3 ✓

Wait! What about √4? It could be 2 or -2. But by convention, √x means the principal (positive) square root. So yes, this is a function.

Not a function!

Consider: "A number whose square is the input"

h(x) = ±√x

Check: Does every input have exactly one output?

  • Input 4 → Output 2 or -2 ✗

No! The input 4 gives two different outputs. This is NOT a function.

📋 How to Solve Most Questions

Step-by-Step: Determining if a Relation is a Function

  1. Identify the inputs and outputs — What goes in? What comes out?
  2. Check each input — Does it have exactly one output?
  3. Look for duplicates — If any input has more than one output, it's not a function
  4. Remember: Different inputs CAN have the same output (that's fine!)
  5. State your conclusion — "Yes, this is a function" or "No, this is not a function"

Step-by-Step: Evaluating a Function

  1. Write down the function — f(x) = [expression]
  2. Replace x with the input value — Put parentheses around it
  3. Simplify — Follow order of operations (PEMDAS)
  4. Write the answer — f(value) = result

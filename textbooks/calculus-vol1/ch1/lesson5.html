Combining Functions

Chapter 1: Functions and Graphs Lesson 5 of 5 ⏱️ 30-40 minutes

🎯 Goal

Learn how to add, subtract, multiply, divide, and compose functions.

In one sentence: You can create new functions by combining existing ones using arithmetic operations and composition.

💡 The Idea

Arithmetic Combinations

Sum and Difference

(f + g)(x) = f(x) + g(x)

(f - g)(x) = f(x) - g(x)

Product and Quotient

(f · g)(x) = f(x) · g(x)

(f / g)(x) = f(x) / g(x), where g(x) ≠ 0

Composition

Composition means applying one function to the output of another:

(f ∘ g)(x) = f(g(x))

Read as "f composed with g" or "f of g of x"

Why This Matters

Combining functions allows us to build complex functions from simple ones. Composition is especially important in calculus for the chain rule.

📝 Example

Example: f(x) = x², g(x) = x + 1

Find (f ∘ g)(x)

Step 1: Apply g first

g(x) = x + 1

Step 2: Apply f to the result

f(g(x)) = f(x + 1) = (x + 1)²

Step 3: Simplify

(f ∘ g)(x) = x² + 2x + 1

📋 How to Solve Most Questions

Step-by-Step: Composing Functions

  1. Identify the order — Which function is applied first?
  2. Substitute — Replace x in the outer function with the inner function
  3. Simplify — Expand and combine like terms
  4. State the domain — Where is the composition defined?

✅ Check Yourself

Test Your Understanding

  • What's the difference between f(g(x)) and g(f(x))?
  • If f(x) = 2x and g(x) = x + 3, find (f ∘ g)(x)
  • When is (f/g)(x) undefined?
  • Is composition commutative?

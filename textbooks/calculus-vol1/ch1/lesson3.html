Graphing Functions

Chapter 1: Functions and Graphs Lesson 3 of 5 ⏱️ 30-40 minutes

🎯 Goal

Learn how to graph functions by plotting points and understanding key features.

In one sentence: Graphing a function means plotting all its input-output pairs on a coordinate plane.

💡 The Idea

What is a Graph?

A graph of a function is the set of all points (x, f(x)) on the coordinate plane. It shows the relationship between inputs and outputs visually.

How to Graph a Function

  1. Choose x-values — Pick several inputs from the domain
  2. Calculate y-values — Find f(x) for each input
  3. Plot points — Mark (x, y) on the coordinate plane
  4. Connect smoothly — Draw a curve through the points

Why This Matters

Graphs help us visualize functions, identify patterns, and understand behavior like increasing/decreasing, maxima/minima, and symmetry.

📝 Example

Example: Graph f(x) = x²

Step 1: Choose x-values

x = -3, -2, -1, 0, 1, 2, 3

Step 2: Calculate y-values

f(-3) = 9, f(-2) = 4, f(-1) = 1, f(0) = 0, f(1) = 1, f(2) = 4, f(3) = 9

Step 3: Plot and connect

Points: (-3,9), (-2,4), (-1,1), (0,0), (1,1), (2,4), (3,9)

This creates a parabola opening upward.

📋 How to Solve Most Questions

Step-by-Step: Graphing a Function

  1. Identify the domain — What x-values are allowed?
  2. Make a table — Calculate f(x) for several x-values
  3. Plot the points — Mark them on the coordinate plane
  4. Connect smoothly — Draw the curve through the points
  5. Check key features — Intercepts, symmetry, behavior

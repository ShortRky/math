Differentiation Rules

Chapter 3: Derivatives Lesson 3 of 7 ⏱️ 35-45 minutes

🎯 Goal

Learn the basic rules for finding derivatives without using the limit definition.

In one sentence: Differentiation rules let you find derivatives quickly without using limits every time.

💡 The Idea

Basic Rules

Constant Rule

d/dx [c] = 0

The derivative of a constant is zero

Constant Multiple Rule

d/dx [c·f(x)] = c·f'(x)

Constants factor out

Sum/Difference Rule

d/dx [f(x) ± g(x)] = f'(x) ± g'(x)

Derivative of sum is sum of derivatives

Why This Matters

These rules save time and make finding derivatives practical for complex functions.

📝 Example

Example: Find d/dx [3x⁴ + 5x² - 7]

Step 1: Apply sum rule

d/dx [3x⁴] + d/dx [5x²] - d/dx [7]

Step 2: Apply constant multiple rule

3·d/dx [x⁴] + 5·d/dx [x²] - 0

Step 3: Apply power rule

3·4x³ + 5·2x = 12x³ + 10x

📋 How to Solve Most Questions

Step-by-Step: Using Differentiation Rules

  1. Identify the structure — Sum, product, chain?
  2. Apply appropriate rule — Use the right rule for each part
  3. Simplify — Combine like terms
  4. Check — Does the answer make sense?
